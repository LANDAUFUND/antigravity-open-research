# ЖУРНАЛ СИСТЕМАТИЧЕСКИХ ОШИБОК И АРТЕФАКТОВ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Журнал нужен для накопления всех причин ложных весовых, силовых и полевых сигналов. Он должен заполняться даже при отрицательных результатах.

## 2. ТИПЫ ОШИБОК

| Код | Тип | Пример |
| --- | --- | --- |
| `ERR-MECH` | механика | касание кабеля, перекос опоры |
| `ERR-THERM` | тепло | конвекция, тепловое расширение |
| `ERR-EM` | электромагнетизм | магнитное притяжение, токовая петля |
| `ERR-ESD` | электростатика | заряд поверхности |
| `ERR-AIR` | воздух | сквозняк, ионный ветер |
| `ERR-SENSOR` | прибор | дрейф нуля, насыщение |
| `ERR-DATA` | обработка | фильтр после просмотра результата |
| `ERR-HUMAN` | оператор | изменение конфигурации без записи |

## 3. ФОРМА ЗАПИСИ

| Поле | Смысл |
| --- | --- |
| Дата | время обнаружения |
| Серия | код опыта |
| Симптом | что увидели |
| Причина | подтвержденная или предполагаемая |
| Проверка | как проверили |
| Влияние | оценка в `Н` или других единицах |
| Решение | исправление |
| Статус | открыт / закрыт / требует повторения |

## 4. ФОРМУЛЫ ОЦЕНКИ

```text
Fошибка ≈ ΔPнабл - ΔPконтроль
Доля_ошибки = |Fошибка| / |ΔPнабл|
```

## 5. ПРАВИЛО ИНТЕРПРЕТАЦИИ

Если систематическая ошибка по масштабу сравнима с заявленным эффектом, серия не может поддерживать гипотезу антигравитации.

## 6. ЧТО НЕЛЬЗЯ ДЕЛАТЬ

Нельзя удалять неудачные серии. Нельзя скрывать артефакт, если он объясняет красивый график. Нельзя переносить результат в публикацию без раздела об ошибках.

## 7. НАУЧНЫЙ ФИЛЬТР

Хороший журнал ошибок повышает надежность работы. Отсутствие ошибок в сложном стенде чаще говорит не об идеальном опыте, а о плохом контроле.

## 8. СЛЕДУЮЩИЕ ДОКУМЕНТЫ

Нужны таблица браковки серий, словарь кодов ошибок и регламент независимого аудита данных.
