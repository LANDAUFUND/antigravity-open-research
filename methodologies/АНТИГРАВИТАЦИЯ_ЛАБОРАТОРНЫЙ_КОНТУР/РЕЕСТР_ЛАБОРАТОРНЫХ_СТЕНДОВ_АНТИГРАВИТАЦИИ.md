# РЕЕСТР ЛАБОРАТОРНЫХ СТЕНДОВ АНТИГРАВИТАЦИИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Реестр перечисляет типы стендов, которые нужно создавать по мере развития направления. Каждый стенд получает код, физическую категорию и критерий измеримости.

## 2. РЕЕСТР

| Код | Название | Категория | Основной сигнал | Статус |
| --- | --- | --- | --- | --- |
| `AG-WEIGHT-001` | Стенд измерения веса | установленная физика | `P`, `ΔP` | проект |
| `AG-EM-001` | Электромагнитная компенсация | инженерная компенсация веса | `F = q(E + v × B)` | проект |
| `AG-MAG-001` | Магнитное давление и левитация | инженерная компенсация веса | `p = B² / 2μ0` | проект |
| `AG-PLASMA-001` | Плазменный контроль | инженерная компенсация веса | ионный ветер / `ωp` | проект |
| `AG-CASIMIR-001` | Микросилы Казимира | установленная физика | `F/A` | проект |
| `AG-SC-001` | Сверхпроводники | установленная физика / риск псевдонауки | магнитная сила | проект |
| `AG-META-001` | Метаматериалы | свойства материала | `ρ`, `σ`, `R` | проект |
| `AG-ANOMALY-001` | Остаточный сигнал | гипотеза после исключений | `Fостаток` | не начинать без базовых стендов |

## 3. ПРАВИЛО ПРИОРИТЕТА

Сначала строятся стенды установленной физики и инженерной компенсации веса. Гипотетический стенд `AG-ANOMALY-001` допустим только после накопления контрольных данных.

## 4. ОБЯЗАТЕЛЬНАЯ ДОКУМЕНТАЦИЯ СТЕНДА

Для каждого стенда нужны паспорт, схема, перечень датчиков, калибровочный протокол, журнал сборки, журнал измерений, контрольная матрица и карта рисков.

## 5. ДАННЫЕ

Каждый стенд должен хранить данные в структуре: `сырые данные`, `обработанные данные`, `калибровки`, `фото/видео`, `версии обработки`, `отчет`.

## 6. ЗАПРЕЩЕННЫЕ СТАТУСЫ

Запрещено ставить статус `ДОКАЗАНА АНТИГРАВИТАЦИЯ`. Разрешенные статусы: `ПРОЕКТ`, `СБОРКА`, `КАЛИБРОВКА`, `КОНТРОЛЬ`, `ПРЕДВАРИТЕЛЬНЫЙ СИГНАЛ`, `НЕ ВОСПРОИЗВЕДЕНО`, `АНОМАЛИЯ ТРЕБУЕТ ПОВТОРЕНИЯ`.

## 7. СЛЕДУЮЩИЙ ШАГ

Первым собирать `AG-WEIGHT-001`, потому что без надежного измерения веса остальные заявления не имеют опоры.
