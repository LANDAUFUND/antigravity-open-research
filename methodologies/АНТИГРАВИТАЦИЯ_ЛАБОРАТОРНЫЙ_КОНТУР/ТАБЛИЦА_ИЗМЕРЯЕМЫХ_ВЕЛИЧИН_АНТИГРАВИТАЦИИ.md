# ТАБЛИЦА ИЗМЕРЯЕМЫХ ВЕЛИЧИН АНТИГРАВИТАЦИИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Документ задает список величин, которые нужно измерять в каждом опыте, чтобы отличать установленную физику и инженерную компенсацию веса от гипотетических и спекулятивных моделей.

## 2. ОСНОВНЫЕ ВЕЛИЧИНЫ

| Код | Величина | Обозначение | Единица | Прибор | Обязательность |
| --- | --- | --- | --- | --- | --- |
| `M-001` | Масса образца | `m` | `кг` | весы / масса-компаратор | обязательно |
| `F-001` | Вес на опоре | `P` | `Н` | датчик силы / весы | обязательно |
| `F-002` | Изменение веса | `ΔP` | `Н` | расчет | обязательно |
| `G-001` | Локальное `g` | `g` | `м/с²` | гравиметр / модель | желательно |
| `T-001` | Температура воздуха | `Tair` | `°C` | термометр | обязательно |
| `T-002` | Температура образца | `Tsample` | `°C` | термопара / ИК | обязательно |
| `H-001` | Влажность | `RH` | `%` | гигрометр | обязательно |
| `P-ATM` | Давление воздуха | `patm` | `Па` | барометр | обязательно |
| `B-001` | Магнитная индукция | `B` | `Тл` | магнитометр | обязательно |
| `E-001` | Электрическое поле | `E` | `В/м` | электростатический датчик | желательно |
| `I-001` | Ток | `I` | `А` | амперметр | обязательно при питании |
| `U-001` | Напряжение | `U` | `В` | вольтметр | обязательно при питании |
| `W-001` | Мощность | `Pel` | `Вт` | ваттметр | обязательно при питании |
| `VIB-001` | Вибрация | `aвибр` | `м/с²` | акселерометр | желательно |
| `FLOW-001` | Поток воздуха | `vair` | `м/с` | анемометр | обязательно для открытого стенда |

## 3. ПРОИЗВОДНЫЕ ВЕЛИЧИНЫ

```text
P = m · g
ΔP = Pдо - Pпосле
η = ΔP / Pдо
pмагн = B² / 2μ0
SNR = |сигнал| / σшума
```

## 4. МИНИМАЛЬНАЯ ЧАСТОТА ЗАПИСИ

| Тип данных | Минимум |
| --- | --- |
| Вес / сила | `10 Гц` для быстрых опытов, `1 Гц` для медленных |
| Температура | `0,2-1 Гц` |
| Токи и напряжения | `1-10 Гц` |
| Магнитное поле | `1 Гц` минимум, чаще при импульсах |
| Видео | непрерывно на ключевые фазы опыта |

## 5. КАЧЕСТВО ДАННЫХ

Каждая запись должна иметь время, единицу измерения, серийный номер прибора, статус калибровки и признак режима: `ФОН`, `КАЛИБРОВКА`, `АКТИВНЫЙ ОПЫТ`, `КОНТРОЛЬ`, `ПОСЛЕОПЫТНЫЙ ДРЕЙФ`.

## 6. ЧТО ПРОВЕРЯЕТ ЛАБОРАТОРИЯ

Лаборатория проверяет, можно ли объяснить `ΔP` через известные величины. Если нет, остаточный сигнал не объявляется антигравитацией, а получает статус `АНОМАЛИЯ, ТРЕБУЮЩАЯ ПОВТОРЕНИЯ`.

## 7. КАКИЕ ДАННЫЕ НУЖНО СОБРАТЬ

Нужно собрать сырые ряды, агрегированные таблицы, графики, журнал исключенных точек, фото стенда, условия среды и полный список действий оператора.

## 8. КАКИЕ ВЫВОДЫ ДЕЛАТЬ НЕЛЬЗЯ

Нельзя делать вывод по данным без единиц, без времени, без калибровки или без контрольной серии.

## 9. НАУКА ПРОТИВ ПСЕВДОНАУКИ

Научная таблица позволяет пересчитать результат другим человеком. Псевдонаучная таблица скрывает исходные данные, меняет единицы и подменяет измерения словами.

## 10. СЛЕДУЮЩИЕ ДОКУМЕНТЫ

Нужно подготовить машинно-читаемый шаблон CSV/TSV для этих величин и словарь кодов ошибок датчиков.
