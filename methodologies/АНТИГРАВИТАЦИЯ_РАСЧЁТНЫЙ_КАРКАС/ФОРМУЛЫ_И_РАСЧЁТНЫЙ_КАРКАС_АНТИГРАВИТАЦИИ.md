# ФОРМУЛЫ И РАСЧЁТНЫЙ КАРКАС ЛАБОРАТОРНОГО ИССЛЕДОВАНИЯ АНТИГРАВИТАЦИИ

Формульный и расчетный каркас для направления Landau Fund "Antigravity Research Initiative".

Статус: лабораторный расчетный документ, не доказательство антигравитации.

Главная логика: антигравитационный метод (antigravity method) нельзя начинать с заявления "антигравитация существует". Его нужно строить как систему проверяемых расчетов:

`вес -> компенсация -> поля -> вакуум -> материалы -> эксперимент -> повторяемость`

Все расчеты ниже выполняются в SI. Если лаборатория работает не на стандартной географической широте/высоте, локальное ускорение свободного падения `g` нужно измерять или брать из геодезической модели, а не автоматически считать равным `9.80665 m/s^2`.

Рабочие константы:

| Обозначение | Значение | Единицы |
|---|---:|---|
| `G` | `6.67430 x 10^-11` | `m^3 kg^-1 s^-2` |
| `g0` | `9.80665` | `m s^-2` |
| `c` | `299792458` | `m s^-1` |
| `h` | `6.62607015 x 10^-34` | `J s` |
| `hbar = h / 2pi` | `1.054571817... x 10^-34` | `J s` |
| `e` | `1.602176634 x 10^-19` | `C` |
| `epsilon0` | `8.8541878188 x 10^-12` | `F m^-1` |
| `mu0` | `1.25663706127 x 10^-6` | `N A^-2` |
| `me` | `9.1093837139 x 10^-31` | `kg` |

Статусы:

- Установленная физика (established physics): классическая механика, электромагнетизм, измеримые материалы, плазма, сверхпроводимость, общая теория относительности как проверенная теория в своем домене.
- Инженерная компенсация веса (weight compensation): реальная подъемная сила, опора, тяга, магнитная/электростатическая/аэродинамическая сила, плавучесть.
- Гипотеза (hypothesis): любые заявления о настоящей антигравитации, снижении гравитационной массы, экранировании гравитации или управлении метрикой.
- Формальная спекулятивная модель (speculative but formalized physics): математически записанная модель, например метрика Алькубьерре (Alcubierre metric), без перехода к лабораторному устройству.

Лабораторное правило: если наблюдается "снижение веса", сначала считать это как `Delta P` на весах и исключать обычные силы: кабели, вибрации, поток воздуха, электростатику, магнитную связь с окружением, температурный дрейф, шум датчика, плавучесть, тягу, механический контакт.

Контекст UAP / НЛО:

- Официальные публикации UAP (unidentified anomalous phenomena) важны как массив наблюдений, документов, сенсорных данных и методологических требований к анализу аномалий.
- На `2026-05-24` официальные американские источники показывают, что часть материалов рассекречивается и публикуется, включая PURSUE-релизы; это не равно доказательству антигравитации, внеземной технологии или управляемой метрики.
- Для лаборатории правильный вывод из UAP-контекста такой: если объект или сигнал выглядит "невозможным", сначала нужно восстановить кинематику, сенсорную геометрию, атмосферу, оптику, radar/IR-артефакты, ошибку трекинга, classified aircraft/drone possibility, плазму, баллистический объект, отражение, монтаж/компрессию видео.
- Эзотерические и культурные интерпретации можно хранить только как слой истории идей и human reports. Их нельзя использовать как физическое доказательство без измеряемых данных.

## 1. Установленная физика: гравитация как базовая точка отсчета

Формула:

```text
F = G m1 m2 / r^2
F = G · m1 · m2 / r²
```

Переменные:

- `F` - сила гравитационного притяжения (gravitational force), `N`.
- `G` - гравитационная постоянная, `m^3 kg^-1 s^-2`.
- `m1`, `m2` - массы тел, `kg`.
- `r` - расстояние между центрами масс, `m`.

Единицы:

`N = kg m s^-2`.

Пример расчета:

Для массы `m = 1 kg` у поверхности Земли можно использовать `m1 = m`, `m2 = M_Earth`, `r = R_Earth`. Приближенно получается та же величина, что и `m g`: около `9.8 N`. Поэтому в лабораторной практике вес тела у поверхности Земли обычно считают через `P = m g`, а формулу Ньютона используют как исходную физическую модель и для проверок масштаба.

Задача лаборатории:

- Считать гравитационную силу, которую нужно компенсировать для каждой тестовой массы.
- Разделять гравитационную силу и обычные силы подвеса, упора, тяги, кабелей, воздушных потоков и магнитных взаимодействий.
- Проверять, не является ли заявленный эффект обычной силой, приложенной к стенду.

Какие данные собрать:

- Масса образца и оснастки.
- Геометрия центра масс.
- Локальное `g`.
- Показания весов до/во время/после воздействия.
- Температура, давление, влажность, вибрации, магнитное поле фона.

Ошибки, которые нельзя делать:

- Нельзя называть уменьшение показаний весов антигравитацией без баланса всех внешних сил.
- Нельзя выводить "экранирование гравитации" из одного измерения веса.
- Нельзя подменять гравитационную массу (gravitational mass) аэродинамической, магнитной или электростатической компенсацией.

## 2. Инженерная компенсация веса

Формулы:

```text
P = m g
E = m g h
P = m · g
E = m · g · h
```

Переменные:

- `P` - вес тела (weight), `N`.
- `m` - масса, `kg`.
- `g` - ускорение свободного падения, `m s^-2`.
- `E` - минимальная механическая энергия подъема, `J`.
- `h` - высота подъема, `m`.

Единицы:

- `P`: `N`.
- `E`: `J = N m`.

Пример расчета:

| Масса | Вес при `g0 = 9.80665 m/s^2` | Минимальная энергия подъема на `1 m` |
|---:|---:|---:|
| `1 kg` | `9.81 N` | `9.81 J` |
| `10 kg` | `98.07 N` | `98.07 J` |
| `100 kg` | `980.67 N` | `980.67 J` |
| `1000 kg` | `9806.65 N` | `9806.65 J` |

Для `m = 100 kg`:

```text
P = 100 x 9.80665 = 980.665 N
E = 100 x 9.80665 x 1 = 980.665 J
```

Это идеальный нижний предел без потерь. Реальная установка потребует больше энергии из-за КПД, нагрева, трения, сопротивления воздуха, потерь в электронике и механике.

Задача лаборатории:

- Сначала поставить цель как компенсацию конкретной силы, например `980.67 N` для `100 kg`.
- Проверять, какая часть силы компенсируется реальным механизмом: опора, тяга, магнитная сила, поток, плавучесть.
- Считать энергетический баланс: входная энергия, механическая работа, тепло, потери.

Какие данные собрать:

- Масса полной системы.
- Сила на весах/тензодатчике с частотой дискретизации и погрешностью.
- Входная мощность `P_in`, энергия `E_in`, температура силовых элементов.
- Перемещение, ускорение, вибрации, контактные силы.

Ошибки, которые нельзя делать:

- Нельзя путать подъемную силу с изменением гравитации.
- Нельзя сравнивать только "до/после" без контроля нуля датчика и механической развязки.
- Нельзя игнорировать КПД: `m g h` - минимум, а не обещанная стоимость подъема.

## 3. Электромагнитная левитация: сила Лоренца и магнитное давление

Электромагнитная левитация (electromagnetic levitation) - это не один эффект, а семейство механизмов: сила на заряды, сила на токи, сила на магнитные диполи, давление поля, вихревые токи (eddy currents), диамагнитное отталкивание, ферромагнитное притяжение, сверхпроводниковый эффект Мейснера и пиннинг магнитного потока. Для antigravity research это критический пласт, потому что электромагнитные силы легко дают ньютонные и килоньютонные нагрузки, то есть могут полностью имитировать "потерю веса".

### 3.1. Отдельный пласт: сила Лоренца (Lorentz force)

Формула:

```text
F = q (E + v x B)
F = q(E + v × B)
```

Для плотности силы в проводящей среде:

```text
f = rho_q E + J x B
```

Для проводника с током:

```text
dF = I dl x B
```

Переменные:

- `F` - сила на заряд или тело, `N`.
- `f` - объемная плотность силы, `N m^-3`.
- `q` - электрический заряд, `C`.
- `rho_q` - объемная плотность заряда, `C m^-3`.
- `E` - электрическое поле, `V m^-1`.
- `v` - скорость заряда, `m s^-1`.
- `B` - магнитная индукция, `T`.
- `J` - плотность электрического тока, `A m^-2`.
- `I` - ток в проводнике, `A`.
- `dl` - элемент длины проводника, `m`.
- `x` или `×` - векторное произведение; направление силы зависит от ориентации скорости/тока и поля.

Единицы:

- `qE`: `C V m^-1 = N`.
- `q(v x B)`: `C m s^-1 T = N`.
- `J x B`: `A m^-2 T = N m^-3`.
- `I dl x B`: `A m T = N`.

Пример расчета 1: отдельный заряд.

Электрон движется со скоростью `v = 1.0 x 10^6 m/s` поперек поля `B = 0.10 T`:

```text
F = e v B
F = 1.602 x 10^-19 x 1.0 x 10^6 x 0.10
F = 1.60 x 10^-14 N
```

Для одной частицы это мало, но в плазме и проводниках число частиц огромно, поэтому суммарные электромагнитные силы становятся макроскопическими.

Пример расчета 2: проводник с током.

Проводник длиной `L = 0.20 m` несет ток `I = 50 A` поперек поля `B = 0.50 T`:

```text
F = I L B = 50 x 0.20 x 0.50 = 5 N
```

Это уже кажущееся изменение массы:

```text
Delta m_app = F / g = 5 / 9.80665 = 0.51 kg
```

То есть обычная сила на провод может выглядеть как "минус полкилограмма" на весах.

Математический смысл:

- Электрическая часть `qE` действует на заряд даже без движения.
- Магнитная часть `q(v x B)` действует только на движущийся заряд и всегда перпендикулярна скорости, если рассматривать чисто магнитную силу.
- В сплошной среде удобнее считать не отдельные заряды, а `J x B`: это основной язык магнитогидродинамики (MHD), плазмы, катушек, шин, токовых петель и силовых проводников.
- Сила Лоренца не нарушает сохранение импульса: реакция уходит в источник поля, катушку, магнит, опору, токовую систему или электромагнитное поле.

Что лаборатория должна проверить:

- Нет ли в установке токовых петель, шин, кабелей, экранов или заземлений, на которые действует `I dl x B`.
- Не появляются ли токи Фуко в алюминии, меди, графите, крепеже, корпусе или столе.
- Не действует ли сила на плазму или ионный ветер через `rho_q E + J x B`.
- Не меняется ли показание весов из-за магнитной связи с арматурой, стальным столом, винтами, датчиком Холла, пружиной или тензодатчиком.
- Как меняется сила при развороте поля `B -> -B`, развороте тока `I -> -I`, повороте установки на `180°`, замене образца на немагнитный муляж.

Какие данные нужно собрать:

- Полная карта `E(x,y,z,t)` и `B(x,y,z,t)`.
- Токи во всех силовых и вспомогательных цепях, включая заземление и экраны.
- Геометрия всех проводников, кабелей, катушек, шин и металлических деталей.
- Частота, фаза, импульсная форма, duty cycle и синхронизация с силовым датчиком.
- Показания независимых датчиков: весы, акселерометр, датчик Холла, Rogowski coil, токовые клещи, тепловизор.
- Контрольные прогоны: без тока, без поля, с разворотом тока, с разворотом поля, с dummy-нагрузкой, с разнесением кабелей.

Ошибки и псевдонаучные выводы:

- Нельзя называть силу Лоренца антигравитацией: это базовая электродинамика.
- Нельзя считать "безопорной" установку, где поле опирается на катушку, магнит, корпус, стол или источник питания.
- Нельзя игнорировать миллиньютоны: для точных весов даже `0.001 N` соответствует примерно `0.10 g`.
- Нельзя рассматривать кабель как "не часть эксперимента": кабель часто является главным источником паразитной силы.

### 3.2. Отдельный пласт: магнитное давление (magnetic pressure)

Формула:

```text
p = B^2 / (2 mu0)
p = B² / 2μ0
F_ideal = p A
```

Более общий язык - тензор напряжений Максвелла (Maxwell stress tensor):

```text
T_ij = (1/mu0) (B_i B_j - 0.5 B^2 delta_ij)
F_i = integral_S T_ij n_j dA
```

Переменные:

- `p` - магнитное давление, `Pa = N m^-2`.
- `B` - магнитная индукция, `T`.
- `mu0` - магнитная проницаемость вакуума, `N A^-2`.
- `F_ideal` - идеализированная оценка силы, `N`.
- `A` - эффективная площадь действия поля, `m^2`.
- `T_ij` - компоненты тензора электромагнитных напряжений, `Pa`.
- `n_j` - компоненты нормали к поверхности интегрирования.
- `delta_ij` - символ Кронекера.

Единицы:

`B^2 / mu0` имеет размерность `Pa`, потому что энергия магнитного поля на единицу объема равна `u_B = B^2 / (2 mu0)`.

Пример расчета:

Для `B = 1 T`:

```text
p = 1^2 / (2 x 1.25663706127 x 10^-6)
p = 397887 Pa
```

Это примерно `3.93 atm` по масштабу давления. Для площади `A = 10 cm^2 = 1.0 x 10^-3 m^2` идеализированная верхняя оценка:

```text
F_ideal = 397887 x 1.0 x 10^-3 = 398 N
Delta m_app = 398 / 9.80665 = 40.6 kg
```

Именно поэтому сильные магнитные поля опасны для интерпретации: они могут дать огромную обычную силу, которая внешне выглядит как изменение веса.

Математический смысл:

- Магнитное поле хранит энергию `u_B = B^2/(2 mu0)`.
- Простое `F = p A` работает только как масштабная оценка для удачной геометрии.
- Реальная сила вычисляется через градиенты поля, энергию системы или интеграл тензора напряжений Максвелла по замкнутой поверхности.
- В неоднородном поле могут возникать как силы, так и крутящие моменты.

Что лаборатория должна проверить:

- Где находится обратная сила: магнит/катушка/криостат/стол/стена камеры.
- Не входит ли силовой датчик в магнитный контур.
- Не намагничены ли элементы весов, крепежа, пружин, экранов, подшипников.
- Не меняется ли сила при замене ферромагнитных деталей на латунь, титан, пластик, керамику.
- Достаточно ли далеко расположены источники поля от весов и электроники.

Какие данные нужно собрать:

- 3D-карта `B` и градиентов `grad B`.
- Модель магнитной цепи и материалы с `mu_r`.
- Силы и моменты на всех опорах, не только на образце.
- Нагрев, сопротивление катушек, охлаждение, вибрации, акустика.
- Проверка весов в том же поле с эталонными грузами и немагнитным муляжом.

Ошибки и псевдонаучные выводы:

- Нельзя выводить "гравитационное экранирование" из силы, пропорциональной `B^2`.
- Нельзя применять `p = B^2/2mu0` как точную силу без геометрии, поверхности интегрирования и материала.
- Нельзя забывать, что магнитная левитация - это обычная инженерная компенсация веса, а не отмена гравитации.

## 4. Квантовый вакуум

Формулы:

```text
E0_mode = (1/2) h f
u_EM = (epsilon0 E^2 + B^2 / mu0) / 2
rho_equiv = u / c^2
```

Переменные:

- `E0_mode` - формальная нулевая энергия одной моды (zero-point energy), `J`.
- `h` - постоянная Планка, `J s`.
- `f` - частота моды, `Hz`.
- `u_EM` - плотность энергии электромагнитного поля, `J m^-3`.
- `E` - электрическое поле, `V m^-1`.
- `B` - магнитная индукция, `T`.
- `rho_equiv` - эквивалентная массовая плотность энергии, `kg m^-3`.

Единицы:

- `h f`: `J`.
- `u`: `J m^-3`.
- `u / c^2`: `kg m^-3`.

Пример расчета:

Для одной электромагнитной моды `f = 10 GHz`:

```text
E0_mode = 0.5 x 6.62607015 x 10^-34 x 10^10 = 3.31 x 10^-24 J
```

Для сильного лабораторного магнитного поля `B = 10 T`:

```text
u_B = B^2 / (2 mu0) = 3.98 x 10^7 J/m^3
rho_equiv = u_B / c^2 = 4.43 x 10^-10 kg/m^3
```

Даже очень сильные лабораторные поля дают крайне малую массу-энергию по сравнению с макроскопическими массами. Это не исключает фундаментальных исследований, но задает масштаб.

Задача лаборатории:

- Исследовать вакуумные и граничные эффекты только через измеримые силы, спектры, шумы, частотные сдвиги и тепловые балансы.
- Разделять квантово-полевые эффекты (quantum field effects) и инженерную тягу/левитацию.
- Проверять, не объясняется ли наблюдение обычной электродинамикой, температурным дрейфом или механикой.

Какие данные собрать:

- Геометрия резонаторов/каверн.
- Частоты, добротность `Q`, спектры шума.
- Температура, вакуум, давление, загрязнение поверхностей.
- Материал, проводимость, шероховатость, заряд поверхностей.

Ошибки, которые нельзя делать:

- Нельзя утверждать "вакуумная энергия = бесплатная энергия".
- Нельзя из наличия нулевых флуктуаций выводить подъемную силу.
- Нельзя смешивать математическую регуляризацию вакуумной энергии с извлекаемой макроскопической работой.

## 5. Эффект Казимира

Эффект Казимира (Casimir effect) - это не "антигравитация", а изменение электромагнитной нулевой энергии из-за граничных условий. В лаборатории это огромный пласт на стыке квантовой электродинамики (QED), физики поверхностей, метрологии нанозазоров, материаловедения, MEMS/NEMS, вакуума, электростатики и статистической обработки слабых сил.

Формула для двух идеальных параллельных проводящих пластин:

```text
F / A = - pi^2 hbar c / (240 a^4)
F/A = -π² ħ c / 240a⁴
```

Переменные:

- `F / A` - давление/сила на единицу площади (Casimir pressure), `Pa`.
- `F` - сила между пластинами, `N`.
- `A` - площадь пластины, `m^2`.
- `hbar` - приведенная постоянная Планка, `J s`.
- `c` - скорость света, `m s^-1`.
- `a` - расстояние между пластинами, `m`.
- Минус означает притяжение пластин в идеальной модели.

Границы применимости:

- Формула относится к идеальным, бесконечным, идеально проводящим, идеально параллельным пластинам при нулевой температуре.
- Реальный эксперимент требует поправок: конечная проводимость, шероховатость, конечная площадь, непараллельность, температура, загрязнение, оксидные слои, остаточные заряды, patch potentials.
- Для сфера-пластина часто применяют приближение близости поверхностей (proximity force approximation, PFA), но оно тоже имеет границы применимости.
- Для диэлектриков и жидкостей используются более общие формулы Лифшица (Lifshitz theory), где входят частотные диэлектрические функции материалов.

Единицы:

`hbar c / a^4 = J m / m^4 = J m^-3 = Pa`.

Пример расчета:

Для `a = 100 nm = 1 x 10^-7 m`:

```text
F/A = -13.0 Pa
```

Для `A = 1 cm^2 = 1 x 10^-4 m^2`:

```text
F = -13.0 x 10^-4 = -1.3 x 10^-3 N
```

Для `a = 10 nm` идеальная формула дает уже около `-1.30 x 10^5 Pa`, но в этом диапазоне резко возрастают требования к параллельности, шероховатости, загрязнению, реальной проводимости и устойчивости пластин.

Масштабная таблица для идеальных пластин:

| Зазор `a` | Давление `F/A` | Сила на `1 cm^2` | Комментарий |
|---:|---:|---:|---|
| `10 um` | `-1.3 x 10^-7 Pa` | `-1.3 x 10^-11 N` | Очень трудно отделить от шума |
| `1 um` | `-1.3 x 10^-3 Pa` | `-1.3 x 10^-7 N` | Микросила |
| `100 nm` | `-13 Pa` | `-1.3 x 10^-3 N` | Уже измеримо, но электростатика критична |
| `10 nm` | `-1.3 x 10^5 Pa` | `-13 N` | Идеальная формула почти наверняка требует сильных поправок |

Физический смысл:

- Казимирово давление возникает не потому, что "вакуум толкает установку вверх", а потому что граничные условия меняют спектр допустимых электромагнитных мод между поверхностями и снаружи.
- Сила является внутренней для системы поверхностей; чтобы получить внешнюю тягу, нужен асимметричный обмен импульсом с внешней средой или полем, а это уже отдельная физическая модель.
- В закрытой механической системе внутренние Казимировы силы не дают безопорного движения центра масс.
- Казимировы эффекты могут быть важны для микромеханики, прилипания (stiction), наномеханических резонаторов и точной метрологии.

Задача лаборатории:

- Измерять микронные/нанометровые силы как квантово-электродинамический эффект, а не как доказательство антигравитации.
- Сравнивать расчет идеальных пластин с поправками на реальный материал.
- Проверять, может ли геометрия создавать измеримый внутренний силовой баланс без ошибочного вывода о внешней подъемной силе.
- Проверять масштаб `a^-4`: если сигнал не следует ожидаемой зависимости от зазора, это повод искать электростатику, загрязнение, дрейф или механику.
- Разделять Casimir force, van der Waals force, capillary force, electrostatic force и chemical adhesion.

Какие данные собрать:

- Расстояние `a` с нанометровой калибровкой.
- Параллельность и шероховатость поверхностей.
- Материал, проводимость, покрытие, загрязнение.
- Электростатический потенциал, patch potentials, температура, вакуум.
- Сила, дрейф датчика, контрольные образцы.
- Историю очистки, плазменной обработки, окисления, хранения и контакта с воздухом.
- Оптические функции материалов, если расчет идет по модели Лифшица.
- Профиль поверхности: AFM/интерферометрия/профилометрия.
- Геометрию держателей, чтобы исключить механическую передачу нагрузки.

Ошибки, которые нельзя делать:

- Нельзя говорить, что эффект Казимира доказывает антигравитацию.
- Нельзя переносить микромасштабную внутреннюю силу пластин на макроскопическую безопорную тягу.
- Нельзя игнорировать электростатические остаточные заряды и загрязнение поверхностей.
- Нельзя подгонять зазор `a` после измерения силы, потому что зависимость `a^-4` делает результат сверхчувствительным к ошибке расстояния.
- Нельзя использовать идеальную формулу для реальных материалов без указания поправок и неопределенности.
- Нельзя утверждать "энергия вакуума извлечена", если нет полного энергетического цикла: сближение, разделение, заряд/разряд, потери, механическая работа.

Лабораторная программа:

1. Построить стенд "сфера-пластина" как более устойчивую геометрию для первичной метрологии.
2. Провести электростатическую калибровку известным напряжением.
3. Измерить residual potential и patch potentials.
4. Снять кривые сила-зазор на нескольких материалах.
5. Повторить с загрязненной и очищенной поверхностью.
6. Сравнить данные с идеальной формулой и моделью реальных материалов.
7. Публиковать только вместе с сырыми кривыми, картой поверхности и моделью ошибок.

## 6. Общая теория относительности

Формула:

```text
G_munu + Lambda g_munu = (8 pi G / c^4) T_munu
Gμν + Λgμν = (8πG / c⁴)Tμν
```

В индексной записи:

```text
G_{mu nu} + Lambda g_{mu nu} = (8 pi G / c^4) T_{mu nu}
G_{μν} + Λg_{μν} = (8πG / c⁴)T_{μν}
```

Переменные:

- `G_{mu nu}` - тензор Эйнштейна (Einstein tensor), кривизна пространства-времени, `m^-2`.
- `Lambda` - космологическая постоянная (cosmological constant), `m^-2`.
- `g_{mu nu}` - метрический тензор (metric tensor), безразмерен в обычной записи интервала.
- `T_{mu nu}` - тензор энергии-импульса (stress-energy tensor), `J m^-3` или `Pa`.
- `G` - гравитационная постоянная.
- `c` - скорость света.

Единицы:

Коэффициент `(8 pi G / c^4) T_{mu nu}` имеет размерность `m^-2`, как и левая часть.

Пример расчета масштаба:

Шварцшильдов радиус (Schwarzschild radius) для массы `1000 kg`:

```text
r_s = 2 G m / c^2 = 1.49 x 10^-24 m
```

Это показывает, насколько ничтожна кривизна пространства-времени от лабораторной массы по сравнению с инженерными размерами. ОТО важна как строгая теория гравитации, но лабораторная "метрическая инженерия" требует энергетических масштабов, которых обычные стенды не создают.

Задача лаборатории:

- Использовать ОТО как язык для проверки заявлений о метрике, массе-энергии и кривизне.
- Считать тензор энергии-импульса для полей, материалов и плазмы только при достаточной математической строгости.
- Разделять локальную ускоренную систему отсчета, гравитационное поле и инженерную тягу.

Какие данные собрать:

- Массы, энергии, плотности энергии, давления.
- Пространственная геометрия источников поля.
- Оценка гравитационного сигнала относительно чувствительности доступных гравиметров/интерферометров.
- Контроль фоновых гравитационных вариаций.

Ошибки, которые нельзя делать:

- Нельзя говорить "поле искривляет пространство" без оценки масштаба `T_{mu nu}`.
- Нельзя выводить warp drive из одной записи уравнений Эйнштейна.
- Нельзя объявлять отрицательную энергию доступной технологией без проверенного физического источника и измеримой плотности энергии.

## 7. Метрика Алькубьерре

Метрика Алькубьерре (Alcubierre metric) - это математическое решение/ансатц в общей теории относительности, вдохновленное идеей "warp drive". Это не лабораторная технология и не доказанная антигравитация. Ее ценность для Landau Fund в другом: она задает строгий язык, на котором можно проверять любые заявления о "двигателе метрики", "пространственном пузыре", "искривлении впереди/позади аппарата" и "безреактивном движении".

Формула:

```text
ds^2 = -c^2 dt^2 + [dx - v_s f(r_s) dt]^2 + dy^2 + dz^2
ds² = -c²dt² + [dx - v_s f(r_s)dt]² + dy² + dz²
```

Переменные:

- `ds^2` - интервал пространства-времени (spacetime interval), `m^2`.
- `c` - скорость света, `m s^-1`.
- `dt`, `dx`, `dy`, `dz` - координатные дифференциалы времени и пространства.
- `v_s` - координатная скорость центра "пузыря" (bubble velocity), `m s^-1`.
- `f(r_s)` - функция формы (shape function), безразмерная.
- `r_s` - расстояние до центра пузыря в выбранной системе координат, `m`.

Один из типовых вариантов расстояния:

```text
r_s = sqrt((x - x_s(t))^2 + y^2 + z^2)
```

Где `x_s(t)` - координата центра пузыря, `m`.

Типовая функция формы:

```text
f(r_s) = [tanh(sigma (r_s + R)) - tanh(sigma (r_s - R))] / [2 tanh(sigma R)]
```

Переменные функции формы:

- `R` - радиус области пузыря, `m`.
- `sigma` - крутизна стенки пузыря, `m^-1`.
- `f(r_s) ≈ 1` внутри пузыря.
- `f(r_s) ≈ 0` далеко снаружи.
- Основные градиенты поля сосредоточены в стенке, где `df/dr_s` велико.

Единицы:

Каждый член в выражении для `ds^2` должен иметь размерность `m^2`.

Математический смысл:

- Внутри идеализированного пузыря локальная область может быть близка к плоскому пространству, но внешние наблюдатели видят движение области за счет сжатия/расширения геометрии.
- Уравнение Эйнштейна требует источник: метрика сама по себе не является устройством. Нужно найти `T_{mu nu}`, который создает эту геометрию.
- В исходной модели возникают области с отрицательной плотностью энергии относительно некоторых наблюдателей. Это связывает модель с экзотической материей (exotic matter) и нарушениями энергетических условий.
- Даже если метрика математически записана, остаются вопросы устойчивости, причинности, горизонтов, управления, включения/выключения, квантовых неравенств и backreaction.

Пример расчета масштаба:

Если формально поставить `v_s = 1 m/s`, то безразмерное отношение:

```text
v_s / c = 3.34 x 10^-9
```

Но даже малый параметр скорости не решает главный вопрос: какой физический `T_{mu nu}` создает такую метрику, выполняются ли энергетические условия, нужна ли отрицательная плотность энергии, стабильна ли конфигурация, можно ли ее включить/выключить и управлять ею.

Пример проверки размерности стенки:

Если `R = 1 m`, `sigma = 10 m^-1`, то характерная толщина стенки порядка:

```text
delta_wall ~ 1 / sigma = 0.1 m
```

Если увеличить `sigma`, стенка становится тоньше, градиенты больше, а требуемые напряжения/энергии обычно растут. Поэтому "резкий пузырь" не бесплатен математически.

Что считать в теоретическом пакете:

```text
g_munu -> Gamma^alpha_munu -> R_munu -> G_munu -> T_munu = (c^4 / 8 pi G)(G_munu + Lambda g_munu)
```

Где:

- `g_munu` - метрический тензор.
- `Gamma^alpha_munu` - символы Кристоффеля.
- `R_munu` - тензор Риччи.
- `G_munu` - тензор Эйнштейна.
- `T_munu` - требуемый тензор энергии-импульса.

Если расчет не доведен до `T_munu`, разговор о "двигателе Алькубьерре" остается картинкой, а не физикой.

Задача лаборатории:

- Хранить метрику Алькубьерре в разделе "математически формализованная, но спекулятивная физика" (speculative but formalized physics).
- Использовать ее как тест на строгость: любая заявка на "метрический двигатель" должна давать метрику, источник `T_{mu nu}`, энергетический баланс и проверяемые предсказания.
- Считать, какие энергетические условия нарушаются или сохраняются.
- Отдельно моделировать аналоговые системы (analogue gravity): оптические метаматериалы, волноводы, жидкости, конденсаты, плазмы. Аналоговая модель может имитировать уравнения для волн, но не означает реальное искривление пространства-времени.
- Привязывать любые UAP/НЛО claims только к измеряемым величинам: скорость, ускорение, тепловой след, спектр, radar cross-section, импульс, энергия, траектория.

Какие данные собрать:

- Математическая форма `f(r_s)`.
- Распределение энергии-импульса, требуемое моделью.
- Знаки и величины плотности энергии.
- Предсказанные измеримые эффекты: фаза интерферометра, гравиметрический сигнал, частотные сдвиги.
- Энергетические условия: weak, null, dominant, strong energy condition.
- Численную устойчивость решения и чувствительность к форме `f(r_s)`.
- Для аналоговых стендов: реальные `epsilon`, `mu`, дисперсия, потери, нелинейность, температура, диапазон частот.
- Для сопоставления с UAP: метаданные сенсоров, оптическая геометрия, radar/IR calibration, атмосферные условия, исходное видео без компрессии.

Ошибки, которые нельзя делать:

- Нельзя называть метрику готовым двигателем.
- Нельзя утверждать FTL-транспорт или антигравитацию без физического источника метрики.
- Нельзя скрывать, что исходные warp-модели связаны с экзотическими условиями, включая отрицательные плотности энергии.
- Нельзя использовать секретность или "закрытые технологии" как замену `T_{mu nu}` и расчету энергии.
- Нельзя приравнивать UAP-релиз к подтверждению Alcubierre drive: официальный статус "unresolved" означает недостаточность идентификации, а не доказательство конкретной физики.
- Нельзя смешивать аналоговую модель распространения волн в материале с реальной управляемой метрикой пространства-времени.

Исследовательская карта:

1. Базовая ОТО: метрика, геодезические, кривизна, энергия-импульс.
2. Исходная модель Алькубьерре: функция формы, пузырь, сверхсветовое координатное движение, экзотическая энергия.
3. Современные модификации: subluminal positive-energy warp concepts, оптимизация энергии, квантовые ограничения.
4. Аналоговые модели: metamaterial analogue, optical metric, effective medium.
5. Лабораторная проверка: не "создать warp", а искать измеряемые ограничения на фазу, поле, силу и энергию.
6. UAP-контекст: использовать как источник задач по реконструкции кинематики, а не как готовое доказательство.

## 8. Инерция и масса

Формулы:

```text
F = m a
E = m c^2
eta_m = m_g / m_i
```

Переменные:

- `F` - сила, `N`.
- `m` или `m_i` - инертная масса (inertial mass), `kg`.
- `m_g` - гравитационная масса (gravitational mass), `kg`.
- `a` - ускорение, `m s^-2`.
- `E` - энергия массы (mass-energy), `J`.
- `eta_m` - отношение гравитационной и инертной массы, безразмерно.

Единицы:

- `m a`: `kg m s^-2 = N`.
- `m c^2`: `kg m^2 s^-2 = J`.

Пример расчета:

Чтобы ускорить `100 kg` с ускорением `0.1 g`:

```text
F = 100 x 0.980665 = 98.07 N
```

Чтобы полностью удерживать `100 kg` от падения:

```text
F = 100 x 9.80665 = 980.67 N
```

Если весы показывают снижение веса на `1%`, это `9.81 N` для `100 kg`. Такой сигнал должен быть сопоставлен с ускорением, внешними силами и систематическими ошибками.

Задача лаборатории:

- Разделять изменение показаний веса, изменение инертной массы и обычное приложение силы.
- Проверять инертную массу динамическими тестами, а гравитационную - статическими/гравиметрическими тестами.
- Сравнивать `m_g` и `m_i` только в контролируемых условиях.

Какие данные собрать:

- Статическое взвешивание.
- Динамический отклик на известную силу.
- Ускорение, вибрации, фазовые задержки, контактные силы.
- Повторяемость при разных ориентациях и нагрузках.

Ошибки, которые нельзя делать:

- Нельзя говорить "масса уменьшилась", если изменились только показания весов.
- Нельзя путать инерциальную развязку, подвес, вибрационную компенсацию и снижение гравитационной массы.
- Нельзя делать вывод по одному объекту без контрольного груза той же массы и геометрии.

## 9. Плазменные модели

Плазма (plasma) - это не "светящийся газ" в бытовом смысле, а коллективная электромагнитная среда из зарядов, полей, волн, столкновений и границ. Для antigravity research это отдельный пласт, потому что плазма может создавать тягу, давление, электростатические силы, магнитогидродинамические силы, ионный ветер, RF-нагрев и сильные сенсорные помехи. Все это может имитировать аномальное изменение веса.

Формула:

```text
omega_p = sqrt(n_e e^2 / (epsilon0 m_e))
ωp = √(ne e² / ε0 me)
```

Переменные:

- `omega_p` - электронная плазменная частота (electron plasma frequency), `rad s^-1`.
- `n_e` - концентрация электронов, `m^-3`.
- `e` - элементарный заряд, `C`.
- `epsilon0` - электрическая постоянная, `F m^-1`.
- `m_e` - масса электрона, `kg`.

Связанные плазменные величины:

```text
f_p = omega_p / (2 pi)
lambda_D = sqrt(epsilon0 k_B T_e / (n_e e^2))
N_D = (4/3) pi n_e lambda_D^3
omega_c,e = e B / m_e
```

Переменные:

- `f_p` - плазменная частота в герцах, `Hz`.
- `lambda_D` - дебаевская длина (Debye length), `m`.
- `k_B` - постоянная Больцмана, `J K^-1`.
- `T_e` - электронная температура, `K` или `eV` при корректной конверсии.
- `N_D` - число частиц в дебаевской сфере, безразмерно.
- `omega_c,e` - электронная циклотронная частота (electron cyclotron frequency), `rad s^-1`.

Единицы:

Подкоренное выражение имеет размерность `s^-2`, поэтому `omega_p` имеет `s^-1`.

Пример расчета:

Для `n_e = 1 x 10^18 m^-3`:

```text
omega_p = 5.64 x 10^10 rad/s
f_p = omega_p / (2 pi) = 8.98 GHz
```

Если `B = 0.10 T`, электронная циклотронная частота:

```text
omega_c,e = e B / m_e
omega_c,e = 1.602 x 10^-19 x 0.10 / 9.109 x 10^-31
omega_c,e = 1.76 x 10^10 rad/s
f_c,e = 2.80 GHz
```

Сравнение `omega_p` и `omega_c,e` показывает, какие волны и режимы возможны: немагнитная плазма, магнитизированная плазма, cutoff, resonance, sheath effects.

Физический смысл плазменной частоты:

- Электроны могут коллективно колебаться относительно более тяжелого ионного фона.
- Ниже `f_p` электромагнитная волна в простой холодной плазме может не распространяться как в вакууме; выше `f_p` поведение меняется.
- `omega_p` зависит от плотности электронов, а не от "антигравитационного потенциала".
- В разрядах у электродов всегда есть sheath-слои, которые могут давать сильные локальные поля и механические силы через ионы.

Плазма как источник обычной тяги:

```text
F_thrust = mdot v_exhaust
P_kin = 0.5 mdot v_exhaust^2
I_sp = v_exhaust / g0
```

Переменные:

- `F_thrust` - реактивная тяга, `N`.
- `mdot` - массовый расход газа/плазмы, `kg s^-1`.
- `v_exhaust` - скорость истечения, `m s^-1`.
- `P_kin` - кинетическая мощность потока, `W`.
- `I_sp` - удельный импульс, `s`.

Если есть выброс массы или импульса в газ/стенки/электроды, это не антигравитация, а обычный перенос импульса.

Задача лаборатории:

- Проверять плазменные и полевые эффекты (plasma and field effects) через измеримую электродинамику: плотность, температуру, частоты, токи, магнитные поля, излучение.
- Считать, может ли плазма создавать тягу, поток, давление, ионный ветер или электростатическую силу.
- Разделять плазменную тягу и гипотезы о гравитационном изменении.
- Проверять, куда уходит импульс: в газ, электрод, стенку вакуумной камеры, магнитную систему, кабель, заземление.
- Делать отдельные режимы: воздух, азот, аргон, вакуум, разное давление, закрытый объем, открытый объем.
- Сравнивать plasma-on, glow-only dummy, RF-heating dummy и thermal dummy.

Какие данные собрать:

- `n_e`, электронная/ионная температура, состав газа.
- Давление, расход газа, вакуум.
- RF/микроволновая мощность, спектр излучения.
- `E`, `B`, токи, напряжения, заземления.
- Вес/тяга на изолированном стенде с dummy-plasma контролем.
- Спектроскопия плазмы для оценки состава и температуры.
- Массовый расход газа, давление по времени, утечки.
- Потенциалы электродов, sheath voltage, floating potential.
- Радиочастотные помехи на весах и датчиках.
- Сила на стенках камеры и электродах, не только на центральном образце.
- Синхронные данные: сила, ток, напряжение, свет, акустика, давление.

Ошибки, которые нельзя делать:

- Нельзя принимать свечение плазмы за новый гравитационный эффект.
- Нельзя игнорировать ионный ветер, нагрев воздуха, акустические силы и электростатику.
- Нельзя использовать плазменную частоту как доказательство антигравитации: это параметр коллективной электродинамики.
- Нельзя называть тягу "безопорной", если поток взаимодействует с воздухом или стенками камеры.
- Нельзя считать вакуумный стенд закрытой системой, если через него идут кабели, охлаждение, RF-питание, газовые линии или утечки.
- Нельзя верить весам рядом с RF-разрядом без проверки электромагнитной совместимости.

Лабораторная программа:

1. Построить плазменный стенд сначала как обычный electric propulsion / plasma actuator test.
2. Измерить thrust на калиброванном маятнике или torsion balance.
3. Сравнить воздух и вакуум, чтобы отделить ion wind.
4. Снять `n_e`, спектр, давление, расход и температуру.
5. Проверить закрытый импульсный баланс: центральный модуль плюс стенки камеры.
6. Только после этого смотреть остаточный сигнал, который не объясняется обычной плазмой.

## 10. Материалы и физико-химические свойства

Материалы и физико-химические свойства (materials and physico-chemical properties) - это не вспомогательный раздел, а основание всей лаборатории. Любая заявка на "антигравитационный материал", "метаматериал", "сверхпроводник", "наноструктуру", "порошок", "керамику", "покрытие" или "сплав" должна быть сначала сведена к измеряемым свойствам: масса, плотность, фаза, состав, микроструктура, проводимость, магнитная восприимчивость, теплопроводность, прочность, сорбция, дегазация, химическая стабильность.

Формулы:

```text
rho_m = m / V
sigma = F / A
kappa = Qdot L / (A DeltaT)
R = rho_e L / A
ρ = m / V
σ = F / A
κ = Qdot · L / (A · ΔT)
R = ρe · L / A
```

Дополнительные связи для лаборатории:

```text
alpha_L = (1/L) dL/dT
epsilon = Delta L / L
E_Y = sigma / epsilon
P_loss = I^2 R
chi_m = M / H
```

Переменные:

- `rho_m` - плотность материала (mass density), `kg m^-3`.
- `m` - масса, `kg`.
- `V` - объем, `m^3`.
- `sigma` - механическое напряжение (stress), `Pa`.
- `F` - сила, `N`.
- `A` - площадь сечения/контакта, `m^2`.
- `kappa` - теплопроводность (thermal conductivity), `W m^-1 K^-1`.
- `Qdot` - тепловая мощность, `W`. Если измеряется теплота `Q` в джоулях, использовать `Qdot = Q / t`.
- `L` - длина/толщина образца, `m`.
- `DeltaT` - разность температур, `K`.
- `R` - электрическое сопротивление, `Ohm`.
- `rho_e` - удельное электрическое сопротивление, `Ohm m`.
- `alpha_L` - коэффициент линейного теплового расширения, `K^-1`.
- `epsilon` - деформация (strain), безразмерная.
- `E_Y` - модуль Юнга (Young's modulus), `Pa`.
- `P_loss` - джоулевы потери, `W`.
- `I` - ток, `A`.
- `chi_m` - магнитная восприимчивость, безразмерная.
- `M` - намагниченность, `A m^-1`.
- `H` - напряженность магнитного поля, `A m^-1`.

Единицы:

- `rho_m`: `kg m^-3`.
- `sigma`: `N m^-2 = Pa`.
- `kappa`: `W m^-1 K^-1`.
- `R`: `Ohm`.
- `alpha_L`: `K^-1`.
- `E_Y`: `Pa`.
- `P_loss`: `W`.

Пример расчета:

Плотность образца `m = 2.7 kg`, `V = 0.001 m^3`:

```text
rho_m = 2700 kg/m^3
```

Напряжение от груза `10 kg` на площади `1 cm^2 = 1 x 10^-4 m^2`:

```text
sigma = 98.07 / 1 x 10^-4 = 9.81 x 10^5 Pa = 0.981 MPa
```

Сопротивление медного проводника с `rho_e = 1.68 x 10^-8 Ohm m`, `L = 1 m`, `A = 1 mm^2`:

```text
R = 1.68 x 10^-8 / 1 x 10^-6 = 0.0168 Ohm
```

Теплопроводность при `Qdot = 10 W`, `L = 0.01 m`, `A = 0.01 m^2`, `DeltaT = 5 K`:

```text
kappa = 10 x 0.01 / (0.01 x 5) = 2 W/(m K)
```

Пример теплового расширения:

Алюминиевая деталь длиной `L = 0.10 m`, `alpha_L = 23 x 10^-6 K^-1`, нагрелась на `DeltaT = 20 K`:

```text
Delta L = alpha_L L DeltaT
Delta L = 23 x 10^-6 x 0.10 x 20
Delta L = 46 x 10^-6 m = 46 um
```

Для весов и микронных зазоров это огромный сдвиг. Он может изменить плечо рычага, контактную силу, зазор Казимира, магнитную силу и показания тензодатчика.

Пример джоулева нагрева:

Проводник с `R = 0.02 Ohm` несет `I = 50 A`:

```text
P_loss = I^2 R = 50^2 x 0.02 = 50 W
```

Эти `50 W` могут создать тепловую конвекцию, расширение, дегазацию, дрейф нуля и ложный "весовой эффект".

Физико-химический смысл:

- Плотность `rho_m` нужна не только для массы, но и для плавучести, центра масс, инерции, акустики и теплопереноса.
- Напряжение `sigma` определяет, не деформируется ли образец или держатель так, что меняет показания датчика.
- Сопротивление `R` и удельное сопротивление `rho_e` определяют токи, нагрев, электромагнитные силы и шумы.
- Теплопроводность `kappa` и теплоемкость определяют задержки: вес может "плыть" минутами из-за температуры.
- Магнитная восприимчивость `chi_m` решает, будет ли образец притягиваться/отталкиваться в градиенте поля.
- Сорбция воды и газов может менять массу на миллиграммы и граммы, особенно у порошков, пористых материалов, полимеров, аэрогелей, керамик.
- Дегазация в вакууме может имитировать тягу или изменение массы.

Материальные классы, которые нужно вести отдельно:

| Класс | Что важно | Главная ловушка |
|---|---|---|
| Металлы | `rho_e`, `kappa`, `alpha_L`, `mu_r`, вихревые токи | Магнитные/индукционные силы |
| Керамики | диэлектрическая прочность, трещины, вакуумная совместимость | Заряд поверхности и хрупкость |
| Полимеры | сорбция воды, creep, outgassing, диэлектрик | Медленный дрейф массы и формы |
| Композиты | анизотропия, слои, проводящие волокна | Неочевидные токовые пути |
| Порошки | влажность, уплотнение, заряд, газовыделение | Масса и плотность зависят от истории |
| Нанопокрытия | толщина, адгезия, оксиды, загрязнение | Поверхностные силы больше объемных |
| Магнитные материалы | `B-H` петля, гистерезис, остаточная намагниченность | Память поля похожа на "аномалию" |
| Сверхпроводники | `Tc`, `Jc`, pinning, криогеника | Мейснер-левитацию путают с антигравитацией |

Задача лаборатории:

- Подбирать материалы, покрытия, керамики, композиты и проводники по реальным нагрузкам.
- Считать механическую прочность, тепловой режим, электрические потери, пробой, усталость, дегазацию.
- Проверять совместимость материалов с вакуумом, криогенной средой, сильными полями и вибрацией.
- Создать паспорт образца (sample passport): происхождение, партия, состав, обработка, покрытие, очистка, хранение, влажность, термоциклы, магнитная история.
- Разделить bulk properties и surface properties: объемный материал может быть известен, а поверхность после обработки полностью меняет электростатику и Казимир.
- Проверять каждую "аномалию веса" на сорбцию/десорбцию воды, испарение растворителя, окисление, дегазацию, thermal creep и контактное трение.

Какие данные собрать:

- Плотность, модуль Юнга, предел текучести, предел прочности, усталость.
- Теплопроводность, теплоемкость, коэффициент теплового расширения.
- Удельное сопротивление, диэлектрическая прочность, магнитная восприимчивость.
- Химическая стойкость, дегазация, загрязнение поверхностей.
- XRF/EDS/XPS/Raman/FTIR данные состава, если материал заявляется как специальный.
- SEM/AFM/профилометрия поверхности.
- Влажность и масса до/после сушки, вакуума, нагрева, криоцикла.
- `B-H` петли и остаточная намагниченность для всех металлических деталей.
- Контактное сопротивление, термо-ЭДС, leakage current.
- Сертификаты поставщика и независимая проверка ключевых свойств.

Ошибки, которые нельзя делать:

- Нельзя считать идеальные свойства материала без температуры, частоты, старения и допусков.
- Нельзя игнорировать микротрещины, контактное сопротивление, тепловые мосты и механический creep.
- Нельзя использовать слово "метаматериал" как замену конкретным измеренным `epsilon`, `mu`, потерям и прочности.
- Нельзя заявлять "новое вещество меняет гравитацию", если не измерены обычные `rho_m`, `rho_e`, `chi_m`, outgassing, влажность и термодеформация.
- Нельзя сравнивать массу горячего образца с холодным без поправки на конвекцию, плавучесть и дрейф весов.
- Нельзя переносить свойства из datasheet в эксперимент без учета партии, обработки и геометрии.

Лабораторная программа:

1. Ввести единый `SAMPLE_ID` для каждого образца.
2. Сначала измерить массу, геометрию, плотность, влажность, сопротивление и магнитную восприимчивость.
3. Затем провести thermal cycle, vacuum cycle, field exposure cycle.
4. После каждого цикла повторить массу и ключевые свойства.
5. Хранить raw data, фото, спектры, сертификаты и условия хранения.
6. Для каждого "аномального" материала создать пару: рабочий образец и физико-химически близкий контрольный муляж.

## 11. Сверхпроводники

Формулы:

```text
Phi0 = h / (2 e)
lambda_L = sqrt(m_e / (mu0 n_s e^2))
F_ideal <= (B^2 / (2 mu0)) A
```

Переменные:

- `Phi0` - квант магнитного потока (magnetic flux quantum), `Wb`.
- `h` - постоянная Планка, `J s`.
- `e` - элементарный заряд, `C`.
- `lambda_L` - лондоновская глубина проникновения (London penetration depth), `m`.
- `n_s` - плотность сверхпроводящих носителей, `m^-3`.
- `B` - магнитная индукция, `T`.
- `A` - эффективная площадь, `m^2`.
- `F_ideal` - верхняя идеализированная оценка силы от магнитного давления, `N`.

Единицы:

- `Phi0`: `Wb = T m^2`.
- `lambda_L`: `m`.
- `F`: `N`.

Пример расчета:

Квант магнитного потока:

```text
Phi0 = 6.62607015 x 10^-34 / (2 x 1.602176634 x 10^-19)
Phi0 = 2.067833848 x 10^-15 Wb
```

Идеальная оценка магнитной силы при `B = 0.5 T`, `A = 0.01 m^2`:

```text
p = 0.5^2 / (2 mu0) = 99472 Pa
F_ideal = 99472 x 0.01 = 995 N
```

Это примерно вес `101 kg`, но только как верхняя оценка по магнитному давлению. Реальная сила зависит от формы поля, пиннинга вихрей (flux pinning), критических токов, температуры и геометрии.

Задача лаборатории:

- Исследовать сверхпроводимость (superconductivity) как платформу для магнитной левитации, сильных токов и стабильных магнитных полей.
- Измерять `T_c`, критический ток `J_c`, критические поля `H_c1/H_c2`, тепловую стабильность и деградацию.
- Проверять любые заявления о "гравитационном экранировании" отдельными контрольными гравиметрическими тестами.

Какие данные собрать:

- Материал и партия сверхпроводника, структура, ориентация зерен.
- Температура, криогенная мощность, тепловые циклы.
- `J_c(T,B)`, `H_c1`, `H_c2`, сопротивление ниже/выше `T_c`.
- Карта поля, сила левитации, механическая реакция опоры магнита.

Ошибки, которые нельзя делать:

- Нельзя считать эффект Мейснера (Meissner effect) антигравитацией.
- Нельзя утверждать гравитационное экранирование по изменению веса рядом с сильным магнитом без исключения магнитных сил.
- Нельзя игнорировать криогенные потоки газа, обмерзание, вибрации компрессора и температурный дрейф весов.

## 12. Метаматериалы

Формулы:

```text
n_eff = +/- sqrt(epsilon_r_eff mu_r_eff)
Z_eff = sqrt(mu_eff / epsilon_eff)
n_eff = ±√(εr,eff μr,eff)
Z_eff = √(μeff / εeff)
```

Переменные:

- `n_eff` - эффективный показатель преломления (effective refractive index), безразмерный.
- `epsilon_r,eff` - эффективная относительная диэлектрическая проницаемость, безразмерная.
- `mu_r,eff` - эффективная относительная магнитная проницаемость, безразмерная.
- `Z_eff` - эффективный импеданс среды (effective impedance), `Ohm` при абсолютных `mu_eff`, `epsilon_eff`.
- Знак `+/-` выбирается по физической ветви решения, потерям, дисперсии и направлению потока энергии.

Единицы:

- `epsilon_r`, `mu_r`, `n_eff` - безразмерные.
- Абсолютные `epsilon`, `mu` имеют `F m^-1` и `H m^-1`.

Пример расчета:

Для обычного диэлектрика с `epsilon_r = 4`, `mu_r = 1`:

```text
n_eff = sqrt(4 x 1) = 2
```

Для электромагнитного metamaterial с отрицательными эффективными параметрами в узкой полосе частот нужно дополнительно измерять потери, дисперсию, устойчивость и границы применимости эффективной среды.

Задача лаборатории:

- Использовать метаматериалы (metamaterials) как способ управлять электромагнитными полями, резонансами, волновым фронтом и локальной плотностью состояний.
- Проверять, меняется ли сила/поле через обычную электродинамику, а не через "гравитационную проницаемость".
- Строить образцы с измеренными S-параметрами, а не с декларативным названием.

Какие данные собрать:

- Геометрия ячейки, период, допуски изготовления.
- `S11`, `S21`, фаза, потери, полоса частот.
- Эффективные `epsilon`, `mu`, `n`, `Z`, анизотропия.
- Нагрев, пробой, механическая прочность, старение.

Ошибки, которые нельзя делать:

- Нельзя переносить "negative index" из оптики/радиофизики на отрицательную массу или отрицательную гравитацию.
- Нельзя утверждать гравитационное экранирование без гравиметрического измерения и модели ОТО.
- Нельзя игнорировать, что эффективные параметры метаматериала обычно частотные, дисперсионные и ограничены геометрией.

## 13. Экспериментальные стенды

Формулы:

```text
F_res = m g - F_comp
Delta P = P_on - P_off
eta_lift = m g h / E_in
SNR = abs(Delta P) / sigma_noise
```

Переменные:

- `F_res` - остаточная нескомпенсированная сила веса, `N`.
- `F_comp` - компенсирующая инженерная сила, `N`.
- `Delta P` - изменение показаний веса/нагрузки между режимами, `N`.
- `P_on`, `P_off` - показания датчика при включенном/выключенном воздействии, `N`.
- `eta_lift` - энергетическая эффективность подъема, безразмерная.
- `E_in` - входная энергия, `J`.
- `SNR` - отношение сигнал/шум, безразмерное.
- `sigma_noise` - стандартное отклонение шума измерения, `N`.

Единицы:

- Силы: `N`.
- Энергия: `J`.
- `eta_lift`, `SNR`: безразмерные.

Пример расчета:

Для `100 kg` вес равен `980.67 N`. Если установка показывает apparent weight reduction `1%`:

```text
Delta P = 9.81 N
```

Если шум весов `sigma_noise = 0.2 N`:

```text
SNR = 9.81 / 0.2 = 49
```

Высокий `SNR` не доказывает антигравитацию: он только говорит, что сигнал выше случайного шума. Нужно отдельно доказать отсутствие систематических ошибок.

Задача лаборатории:

- Строить стенды с контрольными режимами, слепыми проверками и повторяемостью.
- Писать протокол до эксперимента: гипотеза, формулы, ожидаемый масштаб, датчики, контроль, критерий успеха.
- Разделять "наблюдали изменение веса" и "обнаружили новый гравитационный эффект".

Какие данные собрать:

- Сырые данные датчиков с timestamp.
- Схема стенда, масса всех частей, точки контакта, кабели, охлаждение.
- Температура, давление, влажность, вибрации, акустика, электромагнитный фон.
- Входная мощность, токи, напряжения, заземления.
- Фото/видео стенда, калибровки до и после, контрольные грузы.
- Повторения другим оператором и, при возможности, независимой лабораторией.

Ошибки, которые нельзя делать:

- Нельзя публиковать только обработанный график без сырых данных.
- Нельзя менять критерий успеха после эксперимента.
- Нельзя считать "невидимую силу" антигравитацией, пока не закрыт баланс обычных сил.
- Нельзя использовать один стенд без dummy-нагрузки, blind-run и reversal-test.

## Минимальный лабораторный протокол

Каждый расчетный пакет по заявленному antigravity effect должен иметь:

1. Цель в ньютонах: какую силу нужно компенсировать.
2. Массу и геометрию: что именно взвешивается.
3. Источник воздействия: поле, плазма, материал, криогеника, механика.
4. Энергетический баланс: `E_in`, `P_in`, тепло, работа, потери.
5. Модель обычной физики: электромагнитная, тепловая, механическая, аэродинамическая.
6. Гипотетическая часть: что именно не объясняется обычной физикой.
7. Контроли: выключение, dummy, переворот, удаление образца, экранирование, независимый датчик.
8. Сырые данные и код обработки.
9. Повторяемость.
10. Явный вывод: "инженерная компенсация", "необъясненный сигнал", "ошибка", "требует проверки", но не "антигравитация доказана", пока нет строгого подтверждения.
