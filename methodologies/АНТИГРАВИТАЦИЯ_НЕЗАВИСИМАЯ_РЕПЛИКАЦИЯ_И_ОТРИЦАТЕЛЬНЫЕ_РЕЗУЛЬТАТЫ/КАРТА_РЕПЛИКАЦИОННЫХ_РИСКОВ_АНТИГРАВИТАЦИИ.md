# КАРТА РЕПЛИКАЦИОННЫХ РИСКОВ АНТИГРАВИТАЦИИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Карта фиксирует причины, по которым эффект может исчезнуть при повторении, а также причины ложного подтверждения. Она нужна до начала межлабораторного обмена, чтобы не путать реальную невоспроизводимость с плохо перенесенной методикой.

## 2. ОСНОВНЫЕ РИСКИ

| Код | Риск | Контроль |
| --- | --- | --- |
| `RR-001` | разные калибровки весового канала | единый калибровочный груз и журнал |
| `RR-002` | локальная вибрация пола | акселерометр и ночная контрольная серия |
| `RR-003` | скрытый электромагнитный вклад | карта `E`, `B`, токов и заземления |
| `RR-004` | тепловой поток | тепловая камера или массив датчиков температуры |
| `RR-005` | воздух и конвекция | экран, камера, анемометр или вакуумный режим |
| `RR-006` | операторский фактор | слепой порядок режимов |
| `RR-007` | постобработка после результата | hash raw data и предзаданный скрипт |
| `RR-008` | выбор только удачных серий | обязательный реестр всех серий |
| `RR-009` | неодинаковые материалы | паспорт образца и партии |
| `RR-010` | неэквивалентные стенды | таблица сопоставимости стендов |

## 3. ФОРМУЛЫ ОЦЕНКИ

```text
ΔFмежлаб = |Fостаток_1 - Fостаток_2|
Qрепл = 1 - (ΔFмежлаб / max(|Fостаток_1|, |Fостаток_2|))
Rотриц = Nотрицательных / Nвсех
Rартеф = Nобъясненных_артефактов / Nвсех
```

## 4. ПЕРЕМЕННЫЕ

| Переменная | Смысл |
| --- | --- |
| `ΔFмежлаб` | расхождение остаточной силы между лабораториями |
| `Qрепл` | условный коэффициент близости повторения |
| `Rотриц` | доля отрицательных результатов |
| `Rартеф` | доля результатов, объясненных артефактами |

## 5. ЕДИНИЦЫ ИЗМЕРЕНИЯ

`Fостаток` и `ΔFмежлаб` измеряются в ньютонах. Коэффициенты `Qрепл`, `Rотриц`, `Rартеф` безразмерные. Температура, поля, давление, влажность и вибрации указываются в единицах соответствующих приборов с паспортом точности.

## 6. ПРИМЕР ОЦЕНКИ

Лаборатория 1 получила `Fостаток = 0,004 Н`, лаборатория 2 получила `Fостаток = 0,001 Н`.

```text
ΔFмежлаб = |0,004 - 0,001| = 0,003 Н
Qрепл = 1 - (0,003 / 0,004) = 0,25
```

Такое повторение слабое. Нужно проверить различия в датчиках, температуре, электромагнитном фоне и обработке.

## 7. ЧТО ПРОВЕРЯЕТ ЛАБОРАТОРИЯ

- совпадают ли материалы и геометрия образца;
- совпадают ли датчики и их диапазоны;
- сопоставимы ли шумы стендов;
- одинаково ли определены активный и контрольный режимы;
- одинаково ли исключены кабели, воздух, нагрев и вибрация;
- не переносится ли только красивый результат без полного протокола.

## 8. КАКИЕ ДАННЫЕ НУЖНО СОБРАТЬ

- паспорт каждого стенда;
- паспорт каждого образца;
- калибровки датчиков;
- параметры окружающей среды;
- карту полей и заземления;
- все положительные, нулевые и отрицательные серии;
- хэши исходных данных;
- список отличий между лабораториями.

## 9. КАКИЕ ВЫВОДЫ ДЕЛАТЬ НЕЛЬЗЯ

Нельзя объявлять невоспроизводимый эффект «слишком тонким для обычной науки», если не устранены обычные причины расхождения. Нельзя объявлять одно совпадение подтверждением без оценки рисков ложного положительного результата.

## 10. КАК ОТЛИЧАТЬ НАУЧНУЮ КАРТУ РИСКОВ ОТ ПСЕВДОНАУЧНОЙ

Научная карта рисков заранее перечисляет способы провала. Псевдонаучная карта рисков заранее оправдывает любой провал внешними обстоятельствами без измерений.

## 11. СЛЕДУЮЩИЕ ДОКУМЕНТЫ

Нужны таблица сопоставимости стендов, регламент обмена образцами и протокол независимого аудита raw data.
