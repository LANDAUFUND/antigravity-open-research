# ЧЕКЛИСТ ОТСЕЧЕНИЯ ПСЕВДОНАУКИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Чеклист защищает направление от ложных заявлений, рекламных формулировок, эзотерики, неподтвержденных «секретных технологий» и ошибок измерений.

## 2. КРАСНЫЕ ФЛАГИ

| Код | Признак | Действие |
| --- | --- | --- |
| `PF-001` | нет единиц измерения | вернуть на доработку |
| `PF-002` | нет raw data | не принимать вывод |
| `PF-003` | нет калибровки | серия недействительна |
| `PF-004` | нет контроля электромагнетизма | не делать вывод |
| `PF-005` | нет контроля воздуха и тепла | не делать вывод |
| `PF-006` | утверждается доказанная антигравитация | заблокировать публикацию |
| `PF-007` | секретность заменяет формулу | статус `ТРЕБУЕТ ПРОВЕРКИ` |
| `PF-008` | график без файла данных | не принимать |
| `PF-009` | неудачные серии скрыты | требовать полный журнал |
| `PF-010` | термины используются без определения | вернуть на формализацию |

## 3. ОБЯЗАТЕЛЬНЫЕ ВОПРОСЫ

1. Какая сила измерена?
2. В каких единицах?
3. Какой датчик?
4. Когда была калибровка?
5. Где raw data?
6. Какие контроли выполнены?
7. Как исключены кабели, воздух, тепло, магнетизм, электростатика и вибрации?
8. Какой критерий опровержения?

## 4. ЗАПРЕЩЕННЫЕ ПУБЛИЧНЫЕ ФОРМУЛИРОВКИ

- `АНТИГРАВИТАЦИЯ ДОКАЗАНА`.
- `ГРАВИТАЦИЯ ЭКРАНИРОВАНА`.
- `МАССА ИСЧЕЗЛА`.
- `БЕЗОПОРНАЯ ТЯГА ПОДТВЕРЖДЕНА`.
- `ЭТО СЕКРЕТНО, ПОЭТОМУ ФОРМУЛЫ НЕТ`.

## 5. РАЗРЕШЕННЫЕ ФОРМУЛИРОВКИ

- `НАБЛЮДАЕТСЯ ИЗМЕНЕНИЕ ПОКАЗАНИЙ ДАТЧИКА`.
- `РЕЗУЛЬТАТ ОБЪЯСНЕН ИЗВЕСТНОЙ ФИЗИКОЙ`.
- `ОСТАТОЧНЫЙ СИГНАЛ ТРЕБУЕТ ПОВТОРЕНИЯ`.
- `ГИПОТЕЗА НЕ ПОДТВЕРЖДЕНА`.
- `ДАННЫЕ НЕДОСТАТОЧНЫ ДЛЯ ВЫВОДА`.
