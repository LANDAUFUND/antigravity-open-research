## 1. НАЗНАЧЕНИЕ

Этот протокол фиксирует итоговое решение о том, можно ли выпускать пакет `1.0.0` в публичном контуре. Он не является научным доказательством антигравитации.

Антигравитация рассматривается как исследовательское направление, требующее математической формализации, экспериментальной проверки, измеримости и воспроизводимости.

## 2. ВОЗМОЖНЫЕ РЕШЕНИЯ

| Решение | Значение |
|---|---|
| `GO` | Пакет можно выпускать как документальный/методический релиз с указанными ограничениями |
| `GO С ОГРАНИЧЕНИЯМИ` | Выпуск возможен только после явного переноса ограничений в публичные документы |
| `NO-GO` | Выпуск запрещен до закрытия блокирующих замечаний |
| `DEFER` | Решение отложено, требуется дополнительная проверка |

На момент создания документа решение: `NO-GO ПО УМОЛЧАНИЮ`, потому что review-блоки еще не закрыты.

## 4. КРИТЕРИИ GO

Решение `GO` возможно только если:

- все обязательные review-блоки закрыты;
- все блокирующие несоответствия закрыты;
- все существенные ограничения внесены в release notes;
- публичный и приватный GitHub-контуры сверены;
- Bitrix24-папка в group ID `620` сверена с локальными файлами;
- SHA-256 всех файлов совпадает;
- нет секретов и приватных данных;
- нет утверждения доказанности антигравитации;
- не раскрыты патентно чувствительные материалы без решения;
- публикация соответствует CC BY 4.0 только для утвержденных публичных non-code материалов.

## 5. КРИТЕРИИ NO-GO

Решение `NO-GO` обязательно, если:

- хотя бы один review-блок не закрыт;
- есть блокирующее несоответствие;
- не назначен ответственный за научную проверку;
- не проверены данные и хэши;
- обнаружены секреты или приватные данные;
- лицензия неясна;
- патентный риск не оценен;
- есть формулировка, которая подает гипотезу как доказанный результат.
