# ПРОТОКОЛ RELEASE CANDIDATE 1.0.0 НАПРАВЛЕНИЯ «АНТИГРАВИТАЦИЯ»

ДАТА: `2026-05-25`.

СТАТУС: `RELEASE_CANDIDATE`, НЕ ФИНАЛЬНЫЙ РЕЛИЗ.

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Этот пакет фиксирует первый заполненный релиз-кандидат `1.0.0-rc.1` для публичного DOI/архивного слоя направления «Антигравитация». Цель пакета - убрать временные поля из манифеста, связать публичные файлы с фактическими SHA-256, GitHub commit и Bitrix24 file ID, а также отделить техническую готовность архива от научного утверждения о наличии эффекта.

Пакет не утверждает, что антигравитация доказана. Он фиксирует только воспроизводимую проверку состава документов, размера файлов, хэшей, публичных путей и источников.

## 3. СТРОГОЕ ПРАВИЛО МАНИФЕСТА

В строках манифеста не допускаются временные значения вида `ТРЕБУЕТ_*` для полей:

- `РАЗМЕР_БАЙТ`;
- `SHA256`;
- `GITHUB_COMMIT`;
- `BITRIX24_FILE_ID`;
- `DOI_ИЛИ_АРХИВНЫЙ_URL`.

Каждая строка должна проходить проверку фактического локального файла в публичном репозитории. Значение `ПУТЬ_В_ПАКЕТЕ` является относительным путем от корня публичного репозитория `antigravity-open-research`.

## 4. КОМАНДА ПРОВЕРКИ

Проверка выполняется из публичного репозитория:

```bash
python3 tools/АНТИГРАВИТАЦИЯ_ПРОВЕРКА_АРХИВА/ПРОВЕРКА_МАНИФЕСТА_DOI_АРХИВА_АНТИГРАВИТАЦИИ.py \
  --manifest docs/АНТИГРАВИТАЦИЯ_RELEASE_CANDIDATE_1_0_0/МАНИФЕСТ_RELEASE_CANDIDATE_1_0_0_АНТИГРАВИТАЦИИ.csv \
  --base-dir . \
  --strict
```

Ожидаемый результат для перехода к review: `ok: true`, `rows: 5`, `strict: true`, пустой список `errors`.

## 5. ГРАНИЦА RELEASE CANDIDATE И FINAL RELEASE

`1.0.0-rc.1` можно считать технически собранным только после прохождения строгой проверки манифеста. Финальный `RELEASE_1_0_0` нельзя объявлять до завершения пяти проверок:

1. Научная проверка: формулировки не содержат утверждения о доказанной антигравитации.
2. Data review: публичные агрегаты не содержат raw/private данных и не выдают шум за эффект.
3. License review: публичные non-code материалы остаются под `CC BY 4.0`, кодовые утилиты - только с явной software-лицензией.
4. Patent review: публикация не раскрывает патентоспособные инженерные решения до отдельного решения.
5. Security/privacy review: нет секретов, персональных данных, закрытой координации, токенов, cookie, private keys.

## 6. ЧТО НЕЛЬЗЯ ВЫВОДИТЬ ИЗ ЭТОГО ПАКЕТА

Нельзя делать вывод, что антигравитация доказана, что обнаружен новый физический эффект, что лабораторный стенд показал аномальную силу, или что DOI-ready архив заменяет экспериментальную воспроизводимость.

Можно делать только технический вывод: состав документов `1.0.0-rc.1` проверяем, имеет фиксированные хэши, связан с конкретным публичным GitHub snapshot и сверяемыми Bitrix24 file ID.
