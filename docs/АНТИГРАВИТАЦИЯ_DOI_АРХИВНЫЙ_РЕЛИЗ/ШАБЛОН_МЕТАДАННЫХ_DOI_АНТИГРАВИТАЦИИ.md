# ШАБЛОН МЕТАДАННЫХ DOI АНТИГРАВИТАЦИИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ РАЗДЕЛА

Шаблон задает минимальные метаданные для публикации архивного пакета антигравитации в DOI/архиве. Он предназначен для агрегированных данных и методических документов, а не для объявления доказанного эффекта.

## 2. БАЗОВЫЕ ФОРМУЛЫ

```text
CITATION = AUTHORS + YEAR + TITLE + VERSION + DOI
VERSION = MAJOR.MINOR.PATCH
SHA256(file) = hash
```

## 3. РАСШИФРОВКА ПЕРЕМЕННЫХ

- `AUTHORS` - авторы или организация.
- `YEAR` - год публикации.
- `TITLE` - название архивного пакета.
- `VERSION` - версия пакета.
- `DOI` - постоянный цифровой идентификатор.
- `file` - файл пакета.
- `hash` - SHA-256 файла.

## 4. ЕДИНИЦЫ ИЗМЕРЕНИЯ

- Год: `YYYY`.
- Дата: `YYYY-MM-DD`.
- DOI: строка вида `10.xxxx/...`.
- Версия: `MAJOR.MINOR.PATCH`.
- Хэш: SHA-256.

## 5. ПРИМЕРЫ РАСЧЕТОВ

Пример citation string:

```text
Landau Fund. 2026. Antigravity Open Research: Aggregated Series Archive. Version 1.0.0. DOI: ТРЕБУЕТ_ПУБЛИКАЦИИ.
```

Если DOI еще не выдан, поле должно оставаться `ТРЕБУЕТ_ПУБЛИКАЦИИ`, а не заменяться неофициальной ссылкой.

## 6. ЧТО ИМЕННО ДОЛЖНА ПРОВЕРИТЬ ЛАБОРАТОРИЯ

- Название не заявляет доказанную антигравитацию.
- Описание говорит об агрегированных данных и методике.
- Указана версия.
- Указана лицензия.
- Указаны авторство и организация.
- Указан GitHub commit.
- Указаны ограничения интерпретации.

## 7. КАКИЕ ДАННЫЕ НУЖНО СОБРАТЬ

| Поле | Значение |
| --- | --- |
| `TITLE` | ТРЕБУЕТ_ЗАПОЛНЕНИЯ |
| `AUTHORS` | Landau Fund |
| `VERSION` | 1.0.0 |
| `DATE` | ТРЕБУЕТ_ЗАПОЛНЕНИЯ |
| `LICENSE` | CC BY 4.0 |
| `GITHUB_REPOSITORY` | https://github.com/LANDAUFUND/antigravity-open-research |
| `GITHUB_COMMIT` | ТРЕБУЕТ_ЗАПОЛНЕНИЯ |
| `DOI` | ТРЕБУЕТ_ПУБЛИКАЦИИ |
| `KEYWORDS` | antigravity, open research, measurement, replication, negative results |
| `LIMITATION` | Not evidence that antigravity has been proven |

## 8. КАКИЕ ВЫВОДЫ ДЕЛАТЬ НЕЛЬЗЯ

- Нельзя писать в названии DOI "доказательство антигравитации".
- Нельзя указывать DOI до фактической публикации.
- Нельзя скрывать версию и commit.
- Нельзя смешивать публичную лицензию с закрытыми лабораторными материалами.

## 9. КАК ОТЛИЧАТЬ НАУЧНЫЙ РАСЧЕТ ОТ ПСЕВДОНАУЧНОГО ЗАЯВЛЕНИЯ

Научные метаданные указывают предмет, версию, источник, ограничения и лицензию. Псевдонаучные метаданные используют DOI как знак авторитета и убирают ограничения из описания.

## 10. КАКИЕ СЛЕДУЮЩИЕ ДОКУМЕНТЫ НУЖНО ПОДГОТОВИТЬ

- Фактический metadata export для выбранного архива.
- Финальный citation block.
- Список авторов и вкладов.
- ORCID/организационные идентификаторы, если применимо.
- Версию `CITATION.cff` для архивного пакета.
