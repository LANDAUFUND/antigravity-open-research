# Direction Development Protocol

Status: active from 2026-05-25.

The controlling cross-system protocol for the Antigravity direction is maintained in the Landau Fund root repository:

https://github.com/LANDAUFUND/landau-fund/blob/main/docs/ANTIGRAVITY_DIRECTION_DEVELOPMENT_PROTOCOL.md

This file records how that protocol applies inside this public research repository.

## Repository Role

`LANDAUFUND/antigravity-open-research` is the primary public repository for approved Antigravity research materials.

It may contain:

— public methodologies;
— public laboratory-report templates;
— public datasets and aggregate templates;
— public review and release packages;
— public tools;
— public changelog and citation metadata.

It must not contain:

— secrets;
— passwords;
— API keys;
— private contact data;
— partner-sensitive data;
— raw internal coordination;
— patent-sensitive details not approved for release;
— unapproved drafts;
— claims that antigravity has been proven.

## Operating Rule

Every new package added here must have:

— public-safe scope;
— clear status;
— source and authorship clarity;
— correct license notice;
— changelog update;
— review-gate status when relevant;
— no real secrets or private data.

Large release or review packages should include SHA-256 sums and machine-readable registries when traceability matters.

## Default Scientific Status

The default status remains:

`NO-GO / HOLD until human review and nonconformity closure are complete`.

Machine verification, outreach logs, templates, and draft release notes do not prove antigravity and do not authorize a final release.

## Website Relationship

The live public domain is `landau.website`.

This repository may be linked from the website as the public research source of truth. It should not be treated as proof that the website has been deployed or updated.

Website deployment decisions belong to the root protocol and require confirmed production source, hosting, DNS/redirect, HTTPS, and browser verification.
