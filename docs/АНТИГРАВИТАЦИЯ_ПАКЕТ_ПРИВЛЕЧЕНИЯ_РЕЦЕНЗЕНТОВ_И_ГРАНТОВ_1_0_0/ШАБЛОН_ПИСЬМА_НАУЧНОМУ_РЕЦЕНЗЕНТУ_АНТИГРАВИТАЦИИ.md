# ШАБЛОН ПИСЬМА НАУЧНОМУ РЕЦЕНЗЕНТУ АНТИГРАВИТАЦИИ

СТАТУС: `DRAFT / НЕ ОТПРАВЛЕНО`.

ДАТА: `2026-05-25`.

## 1. ТЕМА

```text
Landau Fund: независимая научная проверка материалов по направлению "Антигравитация"
```

## 2. ПИСЬМО

```text
Здравствуйте, [ИМЯ].

Мы ведем в Landau Fund исследовательское направление "Антигравитация" в строгом научном режиме: без утверждения доказанного эффекта, с разделением установленной физики, инженерной компенсации веса, гипотез и спекулятивных моделей.

Сейчас нам нужен независимый научный review пакета release-candidate 1.0.0. Задача рецензента - не подтвердить направление, а проверить корректность формулировок, фальсифицируемость, ограничения, недопустимые выводы и риски преждевременной публикации.

Мы отдельно фиксируем, что решение "NO-GO" является допустимым и ценным результатом проверки.

Публичный пакет содержит методологические документы, шаблоны данных, правила репликации, отрицательных результатов, release-candidate и review-gate. Закрытые лабораторные материалы, raw data, патентно значимые параметры и инфраструктурные доступы не передаются без отдельного основания и NDA.

Если вам потенциально интересно выступить научным рецензентом, мы готовы прислать:

1. краткий индекс материалов;
2. публичные ссылки на GitHub;
3. форму независимости и конфликта интересов;
4. ожидаемый объем review;
5. формат заключения;
6. условия оплаты или грантового участия, если применимо.

Ключевой ожидаемый результат: письменное заключение с одним из статусов:

- APPROVED;
- APPROVED WITH LIMITATIONS;
- REQUIRES FIXES;
- NO-GO.

Будем признательны за ответ, готовы ли вы рассмотреть такую проверку и какой формат работы для вас возможен.

С уважением,
Команда Landau Fund
```

## 3. ЧТО НЕ ПИСАТЬ В ПЕРВОМ ПИСЬМЕ

Нельзя писать:

- что антигравитация доказана;
- что рецензент должен подтвердить направление;
- что секретность заменяет методологию;
- что UAP/НЛО являются доказательством технологии;
- закрытые параметры стендов;
- raw data;
- токены, пароли, внутренние ссылки с доступом;
- патентно значимые раскрытия до решения patent reviewer.

## 4. ПРИЛОЖЕНИЯ К ПЕРВОМУ ПИСЬМУ

Разрешено приложить или дать ссылку:

- публичный GitHub `LANDAUFUND/antigravity-open-research`;
- краткий индекс публичных документов;
- форму независимости;
- форму заключения;
- public/private boundary note;
- список вопросов для scientific review.

Запрещено в первом касании:

- отправлять закрытый private repo;
- отправлять Bitrix24 private folder links;
- отправлять непроверенные raw files;
- отправлять любые секреты доступа.
