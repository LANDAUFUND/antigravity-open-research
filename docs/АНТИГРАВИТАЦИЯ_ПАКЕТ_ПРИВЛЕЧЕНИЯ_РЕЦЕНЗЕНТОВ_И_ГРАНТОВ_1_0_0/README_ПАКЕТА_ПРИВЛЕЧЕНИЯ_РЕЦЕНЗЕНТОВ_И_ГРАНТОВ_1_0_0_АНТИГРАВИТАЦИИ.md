## 1. НАЗНАЧЕНИЕ

Пакет переводит human review из внутренней процедуры в управляемый внешний контур: поиск рецензентов, первичное письмо, критерии независимости, NDA/public-private boundary, учет статусов outreach, бюджет review и подготовку грантового обоснования.

Антигравитация рассматривается как исследовательское направление, требующее математической формализации, экспериментальной проверки, измеримости и воспроизводимости.

Этот пакет не утверждает, что антигравитация доказана. Он не объявляет финальный `RELEASE_1_0_0`, не назначает реальных людей автоматически и не заменяет подписанные заключения специалистов.

## 3. РОЛИ OUTREACH

Минимальный комплект:

- `SCIENTIFIC_REVIEWER`: физика, методология, фальсифицируемость;
- `DATA_REVIEWER`: воспроизводимость, CSV/manifest/SHA-256, единицы измерения;
- `LICENSE_REVIEWER`: CC BY 4.0, attribution, code/non-code boundary;
- `PATENT_REVIEWER`: патентные риски и redaction до публичной выдачи;
- `SECURITY_PRIVACY_REVIEWER`: секреты, персональные данные, public/private boundary;
- `GRANT_REVIEWER`: грантовая упаковка, бюджет, deliverables, отчетность;
- `RELEASE_MANAGER`: решение `GO / GO С ОГРАНИЧЕНИЯМИ / NO-GO / DEFER`.

## 4. ПРАВИЛО СЕКРЕТНОСТИ

Секретность в этом проекте означает законную защиту:

- персональных данных;
- закрытых лабораторных параметров;
- патентно значимых раскрытий до решения patent reviewer;
- raw data, если в них есть приватные сведения или непройденные артефакты;
- ключей, паролей, токенов, инфраструктурных доступов;
- непроверенных технических режимов, которые нельзя выдавать как доказанный результат.

Секретность не используется для обхода научной проверки, сокрытия ошибок, подмены доказательств или утверждений о доказанной антигравитации.

## 5. GRANT READINESS

Для грантового разговора направление должно быть упаковано как открыто-проверяемая исследовательская программа:

1. Научная цель: проверка заявлений о возможных аномальных силах и исключение известных эффектов.
2. Методология: контрольные режимы, слепые проверки, независимая репликация.
3. Данные: публичные агрегаты, SHA-256, manifest, DOI-ready archive.
4. Риски: явное отделение гипотез от доказанных результатов.
5. Выходы: методики, отчеты нулевых серий, открытые шаблоны, рецензии.
6. Ограничения: никаких заявлений о доказанной антигравитации до воспроизводимых результатов.
