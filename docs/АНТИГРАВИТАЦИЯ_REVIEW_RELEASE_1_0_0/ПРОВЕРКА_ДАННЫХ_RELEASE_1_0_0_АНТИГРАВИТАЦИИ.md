# ПРОВЕРКА ДАННЫХ RELEASE 1.0.0 НАПРАВЛЕНИЯ «АНТИГРАВИТАЦИЯ»

ДАТА: `2026-05-25`.

СТАТУС: `REVIEW GATE`, НЕ ФИНАЛЬНЫЙ `RELEASE_1_0_0`.

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Документ задает data review перед финальным `RELEASE_1_0_0`. Цель - проверить, что публикуемые таблицы, агрегаты, словари, схемы и манифест не содержат raw/private данных, не искажают единицы измерения и не превращают шум в заявленный эффект.

## 2. ВХОДНЫЕ МАТЕРИАЛЫ

- CSV-шаблоны измерений и реестров серий.
- Словарь данных измерений и словарь агрегированных данных.
- JSON-схема публичных агрегатов.
- Таблица соответствия Bitrix24/GitHub.
- Release-candidate манифест и strict-отчет проверки.

## 3. ОБЯЗАТЕЛЬНЫЕ ПРОВЕРКИ ДАННЫХ

- Проверить, что public-пакет не содержит raw/private лабораторных журналов.
- Проверить, что агрегаты не раскрывают закрытые параметры стендов, образцов, режимов управления и патентоспособных конфигураций.
- Проверить, что для каждой измеряемой величины указаны единицы, погрешность, частота измерения, метод калибровки и контрольный канал.
- Проверить, что все веса, силы, ускорения, температуры, поля, токи, напряжения, давления и вибрации имеют согласованные единицы СИ.
- Проверить, что нет смешения `масса`, `вес`, `сила`, `ускорение`, `энергия` и `мощность`.
- Проверить, что нулевые и отрицательные серии сохраняются и не вырезаются из агрегатов.
- Проверить, что SHA-256 и размеры файлов совпадают с манифестом.

## 4. КРИТЕРИИ ПРОХОЖДЕНИЯ

Data review может быть закрыт только если публичный набор содержит воспроизводимую структуру данных, но не содержит закрытых raw-записей, персональных данных, секретов, параметров защищенных стендов и неинтерпретируемых чисел без единиц и погрешностей.

## 5. СТОП-УСЛОВИЯ

- В публичный пакет попали raw/private данные.
- Есть численные значения без единиц измерения.
- Нет контрольных каналов для тепла, вибрации, электростатики, магнитных полей, влажности и давления.
- Статистический шум назван эффектом без confidence interval и независимой репликации.
- CSV или JSON не совпадают со словарем данных.

Прохождение data review не означает подтверждения эффекта. Оно означает только, что структура публичных данных пригодна для проверки, воспроизведения, критики и архивирования.
