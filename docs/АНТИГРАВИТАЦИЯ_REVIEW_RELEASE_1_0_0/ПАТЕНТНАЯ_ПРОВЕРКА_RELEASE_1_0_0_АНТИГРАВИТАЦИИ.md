# ПАТЕНТНАЯ ПРОВЕРКА RELEASE 1.0.0 НАПРАВЛЕНИЯ «АНТИГРАВИТАЦИЯ»

ДАТА: `2026-05-25`.

СТАТУС: `REVIEW GATE`, НЕ ФИНАЛЬНЫЙ `RELEASE_1_0_0`.

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. НАЗНАЧЕНИЕ

Документ задает patent review перед финальным `RELEASE_1_0_0`. Цель - отделить открытый научный каркас от потенциально патентоспособных технических решений, режимов стендов, материалов, схем управления и конфигураций, которые нельзя раскрывать до отдельного решения.

## 2. ВХОДНЫЕ МАТЕРИАЛЫ

- Патентная карта направления.
- Открытый и закрытый контур публикации.
- Публичные методики и release-candidate пакет.
- Внутренние лабораторные документы только для проверки рисков раскрытия.

## 3. ОБЯЗАТЕЛЬНЫЕ ПРОВЕРКИ

- Проверить, что public release не раскрывает точные параметры перспективных стендов.
- Проверить, что материалы, режимы, алгоритмы управления, геометрии и схемы не раскрывают патентоспособное решение.
- Проверить, что публикация не мешает будущей заявке из-за преждевременного раскрытия.
- Проверить, что формулы и физические законы не подаются как патентоспособный объект сами по себе.
- Проверить, что prior art search и invention disclosure остаются во внутреннем контуре.
- Зафиксировать решение: публиковать, отложить, передать юристу, удалить чувствительный фрагмент.

## 4. КРИТЕРИИ ПРОХОЖДЕНИЯ

Patent review может быть закрыт только если публичный пакет содержит научно-методическую рамку и агрегированные шаблоны, но не раскрывает защищаемую инженерную реализацию.

## 5. СТОП-УСЛОВИЯ

- В public попали чертежи, режимы, параметры или алгоритмы перспективного стенда.
- В public попали invention disclosure, заявочные формулы, prior art notes или attorney draft.
- Есть риск утраты новизны до подачи заявки.
- Нет решения, что материал действительно должен быть open research.

## 6. РЕШЕНИЕ

| Поле | Значение |
|---|---|
| Статус patent review | НЕ ЗАВЕРШЕНО |
| Решение по раскрытию | НЕ ПРИНЯТО |
| Блокеры | НЕ ПРОВЕРЕНО |
| Ответственный | НЕ НАЗНАЧЕН |
| Дата решения | НЕ ЗАПОЛНЕНО |

## 7. ЗАПРЕЩЕННЫЕ ВЫВОДЫ

Patent review не означает, что существует патентоспособная антигравитационная технология. Он означает только проверку риска преждевременного раскрытия инженерных решений.
