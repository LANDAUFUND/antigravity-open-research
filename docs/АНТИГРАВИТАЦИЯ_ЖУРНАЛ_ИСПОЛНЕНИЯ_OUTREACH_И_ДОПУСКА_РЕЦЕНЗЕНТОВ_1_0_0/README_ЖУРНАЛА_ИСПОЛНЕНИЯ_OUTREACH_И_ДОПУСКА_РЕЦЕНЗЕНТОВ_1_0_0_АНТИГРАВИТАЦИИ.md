# README ЖУРНАЛА ИСПОЛНЕНИЯ OUTREACH И ДОПУСКА РЕЦЕНЗЕНТОВ 1.0.0 АНТИГРАВИТАЦИИ

## НАЗНАЧЕНИЕ

Документ задает операционный слой исполнения outreach для направления `Антигравитация`: отправка первых писем, учет ответов кандидатов, проверка независимости, оформление NDA/COI, допуск к материалам и учет грантовых возможностей.

Антигравитация рассматривается как исследовательское направление, требующее математической формализации, экспериментальной проверки, измеримости и воспроизводимости.

## ГРАНИЦА ПАКЕТА

Пакет не является фактической рассылкой, не назначает рецензентов, не подтверждает согласие кандидатов и не открывает доступ к private/raw/patent-sensitive материалам.

До получения письменного согласия, декларации независимости и решения о допустимом уровне доступа кандидат остается в статусе `НЕ_ДОПУЩЕН`.

## ПРАВИЛО PUBLIC / PRIVATE

Публичный контур может содержать только обезличенные шаблоны, правила и пустые журналы. Реальные имена, адреса, ответы, NDA, COI, условия оплаты, private-ссылки и sensitive-заметки фиксируются только в закрытом контуре и Bitrix24 с подтвержденным якорем.

## КРИТЕРИЙ ПЕРЕХОДА К СЛЕДУЮЩЕМУ ШАГУ

Следующий шаг разрешен только когда в журналах есть проверяемые записи:

- кому отправлено первое письмо;
- через какой канал отправлено;
- какой ответ получен;
- подписан ли NDA;
- закрыта ли COI-проверка;
- какой уровень доступа разрешен;
- какие материалы переданы;
- какое письменное заключение ожидается.

## НАУЧНОЕ ОГРАНИЧЕНИЕ

Этот пакет не является научным результатом и не подтверждает наличие антигравитационного эффекта. Он только задает учетную дисциплину для внешней проверки.
