# ПРОТОКОЛ ОБРАБОТКИ ОТКАЗОВ И МОЛЧАНИЯ КАНДИДАТОВ АНТИГРАВИТАЦИИ

## НАЗНАЧЕНИЕ

Протокол задает порядок обработки отсутствия ответа, отказа, условного согласия или конфликта интересов у кандидатов в рецензенты.

## СТАТУСЫ ОТВЕТА

- `НЕТ_ОТВЕТА`
- `СОГЛАСИЕ_ПРЕДВАРИТЕЛЬНОЕ`
- `СОГЛАСИЕ_ПОСЛЕ_NDA`
- `ОТКАЗ`
- `ОТКАЗ_ИЗ_ЗА_COI`
- `ОТКАЗ_ИЗ_ЗА_СРОКОВ`
- `ОТКАЗ_ИЗ_ЗА_ТЕМАТИКИ`
- `ТРЕБУЕТ_УТОЧНЕНИЯ`

## ПРАВИЛО FOLLOW-UP

Повторное письмо допускается только если:

- первое письмо было отправлено в подтвержденный канал;
- прошло не менее установленного интервала;
- повторное письмо не содержит давления;
- не добавлены private/raw/patent-sensitive материалы;
- в журнал внесена дата повторного контакта.

## ОБРАБОТКА ОТКАЗА

Отказ кандидата фиксируется без негативной оценки. Нельзя делать выводы о научной состоятельности направления только из отказа рецензента.

Минимальная запись:

- роль;
- дата ответа;
- тип отказа;
- можно ли обращаться повторно;
- нужен ли новый кандидат на эту роль;
- сохраняется ли `NO-GO` по release.

## ОБРАБОТКА МОЛЧАНИЯ

Молчание не является согласием. Если после допустимого числа follow-up ответ не получен, кандидат переводится в статус `ЗАКРЫТ_БЕЗ_ОТВЕТА`, а роль возвращается в поиск.

## ЗАПРЕТ ПСЕВДОНАУЧНОЙ ИНТЕРПРЕТАЦИИ

Нельзя использовать:

- отсутствие ответа как доказательство подавления исследования;
- отказ как доказательство правоты гипотезы;
- интерес кандидата как доказательство эффекта;
- грантовый интерес как научное подтверждение.

## СВЯЗЬ С RELEASE 1.0.0

Пока нет письменных заключений и закрытых `NC-001..NC-006`, статус `RELEASE_1_0_0` остается `NO-GO ПО УМОЛЧАНИЮ`.
