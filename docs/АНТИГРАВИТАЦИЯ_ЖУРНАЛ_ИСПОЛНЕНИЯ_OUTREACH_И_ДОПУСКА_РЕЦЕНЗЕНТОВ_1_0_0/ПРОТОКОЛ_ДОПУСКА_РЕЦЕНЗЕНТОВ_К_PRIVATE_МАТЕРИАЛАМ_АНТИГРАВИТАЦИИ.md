# ПРОТОКОЛ ДОПУСКА РЕЦЕНЗЕНТОВ К PRIVATE МАТЕРИАЛАМ АНТИГРАВИТАЦИИ

## НАЗНАЧЕНИЕ

Протокол определяет, когда внешний или внутренний рецензент может получить доступ к закрытым материалам направления `Антигравитация`.

## УРОВНИ ДОСТУПА

1. `PUBLIC_ONLY` — только публичные методики, шаблоны, release candidate и обезличенные материалы.
2. `PRIVATE_REVIEW` — закрытые review-документы без raw data и без patent-sensitive деталей.
3. `RAW_DATA_REVIEW` — ограниченный доступ к исходным данным только после отдельного решения.
4. `PATENT_SENSITIVE_REVIEW` — доступ к патентно-чувствительным материалам только после отдельного legal/patent решения.

## ОБЯЗАТЕЛЬНЫЕ УСЛОВИЯ

До любого доступа выше `PUBLIC_ONLY` должны быть выполнены условия:

- кандидат идентифицирован вне чата и вне публичного репозитория;
- канал связи подтвержден;
- NDA подписан или принято письменное решение, что NDA не требуется;
- COI-декларация получена;
- COI-риск оценен;
- назначен ответственный внутри LANDAU FUND;
- зафиксирован конкретный перечень материалов;
- запрещена передача секретов, токенов, персональных данных и raw data без отдельного решения.

## ЗАПРЕЩЕННЫЕ ДЕЙСТВИЯ

Нельзя:

- передавать private-ссылки до NDA/COI;
- отправлять raw data по открытому каналу;
- раскрывать патентно-чувствительные технические решения до patent review;
- считать молчание кандидата согласием;
- считать участие рецензента доказательством антигравитации;
- объявлять `RELEASE_1_0_0` до письменных заключений и закрытия `NC-001..NC-006`.

## РЕШЕНИЕ О ДОПУСКЕ

Решение о допуске оформляется в CSV-чеклисте и должно иметь один из статусов:

- `НЕ_ДОПУЩЕН`
- `ДОПУЩЕН_PUBLIC_ONLY`
- `ДОПУЩЕН_PRIVATE_REVIEW`
- `ДОПУЩЕН_RAW_DATA_REVIEW`
- `ДОПУЩЕН_PATENT_SENSITIVE_REVIEW`
- `ОТКАЗАНО`

## НАУЧНОЕ ОГРАНИЧЕНИЕ

Допуск рецензента к материалам не является подтверждением эффекта. Это только условие для независимой проверки.
