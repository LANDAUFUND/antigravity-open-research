# СЛОВАРЬ ДАННЫХ ИЗМЕРЕНИЙ АНТИГРАВИТАЦИИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## НАЗНАЧЕНИЕ

Словарь задает поля для машинно-читаемых таблиц измерений. Цель - исключить произвольные подписи, потерю единиц и ручную подгонку данных после опыта.

## ПОЛЯ

| Поле | Тип | Единица | Описание |
| --- | --- | --- | --- |
| `SERIES_ID` | строка | нет | код серии опыта |
| `TIMESTAMP_UTC` | ISO 8601 | нет | время точки измерения в UTC |
| `MODE` | enum | нет | `BACKGROUND`, `CALIBRATION`, `ACTIVE`, `CONTROL`, `DRIFT` |
| `SAMPLE_ID` | строка | нет | код образца или груза |
| `STAND_ID` | строка | нет | код стенда |
| `SENSOR_ID` | строка | нет | код датчика |
| `MASS_KG` | число | `кг` | масса образца |
| `LOCAL_G_M_S2` | число | `м/с²` | локальное ускорение свободного падения |
| `WEIGHT_N` | число | `Н` | вес или нагрузка на опору |
| `FORCE_N` | число | `Н` | сила датчика |
| `DELTA_WEIGHT_N` | число | `Н` | изменение веса относительно базовой линии |
| `TEMP_AIR_C` | число | `°C` | температура воздуха |
| `TEMP_SAMPLE_C` | число | `°C` | температура образца |
| `HUMIDITY_RH_PERCENT` | число | `%` | относительная влажность |
| `AIR_PRESSURE_PA` | число | `Па` | атмосферное давление |
| `MAGNETIC_FIELD_T` | число | `Тл` | магнитная индукция в рабочей точке |
| `ELECTRIC_FIELD_V_M` | число | `В/м` | электрическое поле |
| `CURRENT_A` | число | `А` | ток |
| `VOLTAGE_V` | число | `В` | напряжение |
| `POWER_W` | число | `Вт` | мощность |
| `VIBRATION_M_S2` | число | `м/с²` | вибрационный фон |
| `AIR_FLOW_M_S` | число | `м/с` | скорость воздушного потока |
| `OPERATOR_EVENT` | строка | нет | действие оператора |
| `QUALITY_FLAG` | enum | нет | `OK`, `WARN`, `BAD`, `EXCLUDED` |
| `COMMENT` | строка | нет | краткий комментарий без секретов и персональных данных |

## ПРАВИЛА

Все численные поля пишутся в СИ. Запрещено менять смысл поля между сериями. Если величина не измерялась, поле остается пустым, а причина фиксируется в отчете.

## НЕДОПУСТИМЫЕ ВЫВОДЫ

Таблица измерений не доказывает антигравитацию сама по себе. Она только создает основу для проверки баланса сил, погрешностей и повторяемости.
