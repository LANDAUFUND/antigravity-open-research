# Public Datasets

Approved public datasets belong here.

Do not commit private, sensitive, personal, partner, credential, or raw unreleased data.

## Templates

- `templates/ШАБЛОН_ТАБЛИЦЫ_ИЗМЕРЕНИЙ_АНТИГРАВИТАЦИИ.csv` - row-level measurement table template.
- `templates/РЕЕСТР_СЕРИЙ_ОПЫТОВ_АНТИГРАВИТАЦИИ.csv` - experiment series registry template.
- `templates/СЛОВАРЬ_ДАННЫХ_ИЗМЕРЕНИЙ_АНТИГРАВИТАЦИИ.md` - field dictionary and data rules.

Templates are not experimental results. They define the data structure for future reproducible measurements.
