# ШАБЛОН ОТЧЁТА СЕРИИ ОПЫТОВ АНТИГРАВИТАЦИИ

АНТИГРАВИТАЦИЯ РАССМАТРИВАЕТСЯ КАК ИССЛЕДОВАТЕЛЬСКОЕ НАПРАВЛЕНИЕ, ТРЕБУЮЩЕЕ МАТЕМАТИЧЕСКОЙ ФОРМАЛИЗАЦИИ, ЭКСПЕРИМЕНТАЛЬНОЙ ПРОВЕРКИ, ИЗМЕРИМОСТИ И ВОСПРОИЗВОДИМОСТИ.

## 1. ИДЕНТИФИКАЦИЯ

| Поле | Значение |
| --- | --- |
| `SERIES_ID` | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |
| Дата | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |
| Стенд | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |
| Оператор | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |
| Версия протокола | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |

## 2. ГИПОТЕЗА

Формулировать только проверяемую гипотезу. Пример: `при включении режима X измеренная нагрузка на опору изменится на величину ΔP, превышающую шум и не объясненную известными силами`.

## 3. КОНФИГУРАЦИЯ СТЕНДА

Описать образец, датчики, питание, кабели, подвесы, экраны, температуру, давление, влажность и расположение источников поля.

## 4. КАЛИБРОВКА

Указать контрольные грузы, нулевой дрейф, погрешность, статус датчиков и файлы калибровки.

## 5. ДАННЫЕ

| Файл | Назначение | Статус |
| --- | --- | --- |
| `RAW_DATA_FILE` | сырой временной ряд | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |
| `CALIBRATION_FILE` | калибровка | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |
| `VIDEO_FILE` | видеофиксация | `ТРЕБУЕТ ЗАПОЛНЕНИЯ` |

## 6. РАСЧЁТ

```text
P = m · g
ΔP = Pдо - Pпосле
Fостаток = ΔP - Fизв
SNR = |Fостаток| / σшума
```

## 7. КОНТРОЛЬНЫЕ ОПЫТЫ

Перечислить выполненные контроли из матрицы: макет, отключенное питание, перестановка кабелей, контроль воздуха, немагнитный образец, независимый датчик.

## 8. ОШИБКИ И АРТЕФАКТЫ

Заполнить журнал систематических ошибок. Если ошибка сравнима с эффектом, серия не поддерживает сильный вывод.

## 9. ВЫВОД

Разрешенные формулировки:

- `ЭФФЕКТ НЕ НАБЛЮДАЛСЯ`.
- `ЭФФЕКТ ОБЪЯСНЕН ИЗВЕСТНОЙ ФИЗИКОЙ`.
- `ПРЕДВАРИТЕЛЬНЫЙ СИГНАЛ, ТРЕБУЕТ ПОВТОРЕНИЯ`.
- `АНОМАЛИЯ, ТРЕБУЕТ НЕЗАВИСИМОЙ ПРОВЕРКИ`.

Запрещенная формулировка без строгой независимой проверки: `АНТИГРАВИТАЦИЯ ДОКАЗАНА`.

## 10. ПУБЛИКАЦИОННЫЙ СТАТУС

Указать: `ВНУТРЕННИЙ`, `ГОТОВИТСЯ К ПУБЛИКАЦИИ`, `ПУБЛИЧНЫЙ`, `НЕ ПУБЛИКОВАТЬ`.
